# 3645 達邁 深度研究報告

## 一句話摘要

達邁科技（3645）作為台灣PI薄膜龍頭，正積極轉型至AI眼鏡、先進封裝、電動車及軍用無人機等高成長非軟板應用，成功切入Meta供應鏈。儘管短期月營收受市場波動影響，但隨著5億元新台幣T6產線擴建預計於2026年下半年開出，整體產能將提升20-25%，配合折疊手機、AI眼鏡等需求爆發，營運有望在2026年進入高速成長期。

## 公司概覽

達邁科技成立於2000年，是台灣最大的聚醯亞胺（Polyimide, PI）薄膜供應商，全球市佔率排名第四。其核心產品PI薄膜因具備高耐熱性、絕緣性、耐化學性與優異的機械強度，廣泛應用於高科技產業。

**核心產品及應用包含：**
*   **聚醯亞胺薄膜 (PI Film)**：主要應用於軟性印刷電路板（FPC）、半導體封裝、液晶顯示器（LCD）、OLED、5G通訊等，是FPC不可或缺的材料。其中，**黑色PI膜約佔營收的7~8成**。
*   **透明PI薄膜 (Transparent PI Film)**：主要用於折疊手機、AR眼鏡、OLED顯示器，已成功進入Meta供應鏈，應用於眼球追蹤及近眼顯示器功能。
*   **耐高溫與散熱材料**：應用於電動車（EV）、AI晶片、航太工業。更高厚度及更高導熱係數的石墨散熱片產品已接近量產。高導熱PI產品已成功導入軍用無人機的散熱應用。
*   **半導體先進封裝材料**：已開始向半導體廠供應先進封裝製程所需關鍵材料，並於2025年第四季起小量出貨，正式跨足高門檻半導體材料市場。

**營收結構（2025年前三季累計）：**

| :----------- | :------- |
| 軟板應用     | 56%      |
| 非軟板應用   | 44%      |
| **合計**     | **100%** |

達邁的生產基地主要位於台灣新竹新埔及苗栗銅鑼廠。公司正積極擴建其T6產線，預計將於2026年第三季至第四季陸續開出，屆時整體PI產能將提升約20%至25%。

## 核心競爭優勢

1.  **市場領先地位與技術實力**：達邁是台灣PI薄膜龍頭，全球市佔率第四，證明其在PI材料領域深厚的技術累積和量產能力。具備黑色PI膜、透明PI薄膜、高導熱PI產品的自主開發與製造實力。
2.  **多元化高價值應用拓展**：成功將產品應用從傳統軟板延伸至高成長、高毛利的非軟板領域，包括Meta AI眼鏡、半導體先進封裝、電動車（EV）高導熱材料、軍用無人機散熱等，降低對單一產業的依賴並提升獲利潛力。
3.  **關鍵供應鏈整合**：透明PI薄膜已成功打入Meta AI眼鏡供應鏈，並開始向台灣半導體大廠供應先進封裝材料，顯示其產品已達到國際一線客戶的嚴苛標準，具備高進入門檻的優勢。
4.  **前瞻性產能擴充**：計畫投入5億元擴建T6產線，預計2026年下半年投產，將整體PI產能提升20-25%，有效解決短期產能限制，為未來爆發性需求做好準備。
5.  **高額研發投入**：2025年前三季研發費用達新台幣1.42億元，持續投入高階絕緣、散熱及半導體材料開發，確保技術領先地位並不斷推出符合市場趨勢的新產品。

### 產業趨勢

1.  **柔性與可折疊電子產品的普及與超薄/透明PI薄膜需求**
    *   **趨勢**：智慧型手機製造商持續商業化折疊式設備，柔性顯示器市場預計到2030年複合年成長率超過12%。市場對超薄（10µm以下）和無色/透明PI薄膜的需求日益增長，要求透光率在450奈米波長下超過85%，並具備超過10萬次折疊循環的機械完整性。
