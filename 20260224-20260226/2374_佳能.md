# 2374 佳能 深度研究報告

## 一句話摘要
佳能（2374）已成功轉型為AI視覺應用與智慧光學解決方案供應商，受惠於機器人、無人機及車載鏡頭等高成長新興領域，2025年營收達99.1億元創8年新高，預期2026年將突破百億大關並挑戰獲利新高，公司積極擴充產能與技術投資，成長動能強勁。

## 公司概覽
佳能企業深耕影像光學技術，已從傳統數位相機代工轉型為結合AI、車用與機器人視覺的智慧光學解決方案供應商，提供ODM服務。

**核心產品與服務：**
*   **影像產品：** 360度環景攝影機、複合式相機，應用於運動攝影、串流直播、VR、智慧零售。
*   **光學模組：** 涵蓋醫療影像、AR/VR與無人機視覺系統。
*   **車載與移動產品：** 由後裝市場延伸至前裝供應鏈，日系車載鏡頭產品預計2026年量產。
*   **AI影像與機器視覺：** 針對機器人視覺、無人機影像模組、智慧城市光學提供解決方案。
*   **核心技術：** 內建神經網路引擎的新一代SoC研發、混合式防手震（OIS+EIS）、整合「熱成像」、「紅外線」與「雷射測距」的「三光一體」多重光學模組。
*   **NRE（委託開發服務）：** 大幅挹注NRE收入，顯示已轉型為光學方案商，參與客戶前端設計。

佳能製造廠區分佈於中國東莞、北越及台灣。

## 核心競爭優勢
1.  **策略轉型成功：** 從傳統數位相機代工廠，成功轉型為AI視覺應用與智慧光學解決方案供應商，掌握邊緣AI、機器人視覺及車載光學等高成長趨勢。
2.  **先進光學與AI技術整合：** 聚焦內建神經網路引擎的新一代SoC研發，強化影像品質與AI辨識效能；具備混合式防手震及整合「熱成像」、「紅外線」與「雷射測距」的「三光一體」多重光學模組技術，提供差異化解決方案。
3.  **NRE服務模式：** 大幅挹注NRE收入，顯示佳能已具備參與客戶前端設計的「光學方案商」能力，加強與客戶的合作深度與黏著度。
4.  **彈性生產與全球佈局：** 東莞、北越、台灣三地生產基地彈性配置，有效規避地緣政治風險，確保供應鏈韌性。
5.  **策略合作與生態系整合：** 與美國Agility Robotics（雙足人形機器人公司）合作，並與鑫蘊林科（Linker Vision）合作，兩家公司均獲Nvidia入股，有助於佳能深化與Nvidia的合作關係，並將AI攝影機方案輸出全球。

### 產業趨勢
1.  **AI與深度學習的深度融合：**
    *   **具體影響：** AI與深度學習算法正大幅提升電腦視覺系統的準確性與效率。佳能積極研發內建神經網路引擎的SoC，將受惠於AI驅動的圖像穩定、物體追蹤、深度映射等高階應用，特別在360度環景攝影機中AI拼接算法的應用。
2.  **車用影像技術的提升與ADAS普及：**
    *   **具體影響：** 自動駕駛與ADAS技術的發展對車載鏡頭的需求不斷擴大，且對影像精度要求大幅提升。佳能已切入前裝供應鏈，日系車載鏡頭預計2026年量產，符合ADAS對高畫素、多鏡頭系統及AI影像處理的需求。
3.  **邊緣AI與智慧光學模組的發展：**
    *   **具體影響：** 邊緣AI晶片效能提升，使推論任務轉移至地端執行，實現低延遲、高隱私的即時決策。佳能聚焦SoC研發，使其智慧光學模組內建AI處理能力，為製造、醫療、零售等產業的AI Box提供解決方案。

**對佳能而言的具體機會和威脅：**
*   **機會：**
    *   AI視覺解決方案市場高成長。
    *   ODM模式彈性與技術整合能力，可與更多品牌客戶合作。
    *   新一代SoC研發，提升產品差異化。
    *   日系車載鏡頭2026年量產，打入前裝供應鏈。
    *   人形機器人視覺應用潛力巨大（與Agility Robotics合作）。
*   **威脅：**
    *   市場競爭激烈，眾多廠商佈局AI與車用領域。
    *   技術快速迭代，需持續高額研發投入。
