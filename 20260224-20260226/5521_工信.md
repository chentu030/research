# 5521 工信 深度研究報告

**產生時間：2026-03-06 14:11**

## 一句話摘要

工信(5521) 作為台灣基礎建設與營造工程領導廠商，受惠於政府公共工程預算成長及AI相關資料中心、能源轉型等新興建設需求，在手案量超過千億元，2025年營收與獲利顯著改善，2026年營運展望樂觀。然而，高本益比與營建認列進度風險仍需密切關注。

## 公司概覽

工信工程股份有限公司（5521）是一家主要提供公共項目建設服務的台灣公司，核心業務廣泛，涵蓋土木、建築、海洋工程及鋼結構製造。

**主要業務與核心產品/服務：**

*   **土木工程**：道路、橋樑、隧道、碼頭、捷運、機場、鐵路改造、高速公路等。
*   **建築工程**：商業與住宅建築。
*   **海洋工程**：電廠改造、擴建規劃。
*   **地質鑽探工程**。
*   **鋼結構與建築材料**：鋼結構設計、製造、安裝，以及各種特殊建築材料（大型建築鋼構、橋樑結構、金屬板材）的生產與銷售。
*   **材料買賣與進出口**。

**營收結構 (2025年數據推估)：**

| 業務類別 | 佔營收比例 | 備註                               |
| :------- | :--------- | :--------------------------------- |
| 建造合同 | 約 98.3%   | 包含土木、建築、海洋工程等主要營造收入 |
| 房地產相關 | 少於 2%    | 佔比相對較小                       |

**製造基地與營收貢獻：**

主要生產據點位於台灣（高雄、台中等地），配備現代化鋼構加工設備。公司亦在中國大陸設有據點，並參與東南亞基礎建設項目。
2024年台灣地區營收為 70.99 億新台幣，佔比 100%。目前無詳細的各製造基地或海外據點營收貢獻數據。

**所屬細分產業與相關題材：**

*   **細分產業**：「基礎建設」、「營造工程」、「建材營造」。
*   **相關題材**：公共工程、基礎建設、土木與建築工程、鋼結構、AI智能基礎建設、能源轉型。

## 核心競爭優勢

1.  **豐富的大型公共工程經驗**：在道路、橋樑、隧道、碼頭、捷運等領域累積深厚實力，具備承接複雜大型專案的能力。
2.  **垂直整合能力**：不僅提供工程服務，同時也是重要的鋼結構及建築材料生產商，有助於成本控制與工程品質管理。
3.  **在手案量龐大**：目前手中在建工程量超過千億元，為未來數年營收提供穩定基礎。
4.  **市場深耕與法規熟悉度**：作為台灣本土企業，對當地法規、環境及市場需求有深刻理解，有助於專案執行效率及風險控管。

## 一句話摘要
工信(5521)受惠於台灣公共建設預算增加及AI相關基礎設施需求，在手訂單逾千億元，推動2025年營收與獲利顯著成長，2026年成長動能強勁，但其股價高本益比與營建專案認列進度仍為關鍵風險。

## 公司概覽
工信工程股份有限公司（5521）為台灣領先的基礎建設與營造工程公司，主要業務聚焦於公共建設項目。

**主要業務與核心產品/服務：**
*   **土木工程**：道路、橋樑、隧道、碼頭、捷運、機場、鐵路改造、高速公路等大型專案。
*   **建築工程**：商業與住宅建築營造。
*   **海洋工程**：電廠改造與擴建規劃等。
*   **地質鑽探工程**。
*   **鋼結構與建築材料**：提供鋼結構設計、製造及安裝服務，並生產銷售大型建築鋼構、橋樑結構及金屬板材等建築材料。
*   **材料買賣與進出口**：工程相關材料的貿易業務。

**營收結構 (2025年推估)：**

| 業務類別   | 佔營收比例 | 備註                 |
| :--------- | :--------- | :------------------- |
| 建造合同   | 約 98.3%   | 主要營造工程收入     |
| 房地產相關 | 少於 2%    | 比例較小，非核心業務 |
