# 3017 奇鋐 深度研究報告

## 一句話摘要
奇鋐（3017）作為AI伺服器液冷散熱解決方案的領導者，受惠於AI晶片功耗提升帶動的強勁需求與技術領先優勢，營收及獲利能力正迎來爆發性成長，預期在2026年將持續受惠新平台與產能擴張，營運展望極為樂觀。

## 公司概覽
奇鋐主要從事散熱解決方案的研發、製造與銷售，其核心產品和服務聚焦於AI伺服器的高階散熱技術。隨著AI運算需求激增，液冷散熱已成為公司最關鍵的成長動能。

**核心產品與服務：**
*   **液冷散熱模組**：水冷板（Water-cooling plate）、快速接頭（Quick Disconnect, UQD）、分歧管（Manifold）等關鍵零組件，以及機櫃級散熱解決方案。主要應用於NVIDIA的GB200、GB300、Vera Rubin系列GPU，Google的TPUv8，AWS的Trainium等ASIC及LPU高功耗晶片。
*   **傳統散熱模組**：應用於PC及其他電子產品。
*   **機殼與機構件**：伺服器機構件及其他電子產品機殼。
*   **風扇馬達**：提供散熱風扇產品。
*   **Microchannel lid**：配合特定客戶需求開發。

**營收結構概覽（預估與成長動能）：**

| 產品/業務類別       | 說明與預期營收比重/成長 |
| :------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **AI伺服器液冷散熱** | 公司最大成長引擎。法人預估，2026年液冷業務營收年增率有望超過**80%**。另有外資預期，液冷產品在總營收中的佔比到2027年有望達到**20%至25%**。高技術門檻的液冷分歧管（Manifold）業務，預期在2026年出貨占比可望提升至**5%～10%**，將為毛利帶來第二波結構性成長。 |
| **伺服器業務**      | 包含散熱與機構件，是公司最大的成長動能。伺服器相關營收比重預計從2024年的約**36%**增至2025年的**45-50%**，驅動利潤率上升。液冷散熱營收比重從2024年上半年的約**3%**，提升至2024年全年的**5-10%**，並在2025年達到**15-20%**或更高。                      |

**製造基地與產能擴充：**
*   **越南廠**：正積極擴充產能，預計在2026年將擴充**50%**，以支應不斷增加的水冷板等高階散熱產品需求。越南三期廠房預計於2026年中前後大幅開出。
*   **中國大陸廠**：亦有擴產計畫以應對未來訂單需求。

**所屬細分產業與相關題材：**
*   **細分產業**：散熱模組產業、AI伺服器散熱、資料中心基礎設施。
*   **相關題材**：AI液冷散熱、高效能運算（HPC）、節能減碳、AI軍備競賽。

## 核心競爭優勢
1.  **AI液冷散熱技術領先者**：奇鋐在AI伺服器液冷散熱領域具備領先地位，擁有完整的產品組合，涵蓋3D VC、風扇、機殼、冷板、分流管、冷卻液分配單元（CDU）等，尤其在水冷板、分歧管、快速接頭等關鍵零組件方面具備高技術壁壘。
2.  **成功切入頂級AI平台供應鏈**：已成功切入NVIDIA GB200/GB300平台，並為其生產冷板模組。新產品VR200、Trainium3水冷產品及Google TPUv8水冷產品亦將陸續量產出貨。
3.  **穩固的客戶基礎與高市佔率**：在北美四大雲端服務供應商（CSP）業者中，水冷散熱模組市占率已達**40%至50%**，是NVIDIA GB系列機櫃的核心供應商，同時也是非NVIDIA陣營AI ASIC伺服器水冷板的重要供應商。
4.  **積極且前瞻的產能擴張**：透過擴充越南廠及中國大陸廠產能，尤其針對高階液冷產品，確保能滿足市場強勁的成長需求。例如，越南廠水冷板模組月產能預計從2024年的**11.5萬組**擴大至2025年的**30萬組**以上。
5.  **毛利率結構性改善**：隨著高單價、高毛利的AI伺服器水冷產品比重持續提升，帶動公司整體毛利率顯著向上，預期2026年上半年毛利率有望達**28%至29%**。

*   **資料中心液冷市場**：預計將從2024年的**49.8億美元**成長到2030年的**108.6億美元**，複合年均成長率（CAGR）為**13.86%**。
