# 6285 啟碁 深度研究報告

## 一句話摘要
啟碁 (6285) 身為緯創集團旗下網通設備設計製造大廠，受惠於全球網路基礎設施升級、低軌衛星應用爆發及車用電子智慧化浪潮，2025年營收達新台幣1,102.5億元創歷史新高，第四季毛利率更躍升至14.1%，為近十年單季新高。在Wi-Fi 7、800G高階交換器及低軌衛星使用者終端等高毛利產品組合優化下，啟碁預期2026年營收及獲利將實現雙位數成長，法人預估EPS上看新台幣8.97元，展現顯著成長動能。

## 公司概覽
啟碁科技（Wistron NeWeb Corp., WNC）成立於1996年，是緯創集團旗下的網通設備設計製造大廠（ODM）。其產品佈局涵蓋所有需要「無線訊號傳輸」的設備。

**核心產品/服務：**
*   **企業與衛星通訊 (Networking)**：提供企業級交換器（如800G高階交換器）、5G小型基地台、非同步軌道衛星寬頻設備以及低軌衛星（LEO）用戶端天線等。啟碁是SpaceX（星鏈）用戶端產品的主要供應商，提供用戶端設備和天線等產品，並被視為美系低軌衛星客戶地面設備站的獨家供應商。
*   **車用電子 (Automotive)**：產品包含車用通訊模組、毫米波雷達、車內與車用攝影機，以及充電樁通訊裝置。
*   **智慧家庭 (Connected Home)**：主要產品有家用路由器、寬頻閘道器（PON）及智慧機上盒。

**營收結構 (2025年產品組合預估)**

| 業務板塊     | 營收佔比 |
| :----------- | :------- |
| 網通 (Networking) | 51%      |
| 車用 (Automotive) | 32%      |
| 智慧家居 (Connected Home) | 17%      |

**製造基地：**
啟碁的製造基地包括越南和墨西哥。公司正積極擴充越南製造基地，預計2026年將成為其全球最大的生產據點，產值可能貢獻總體營收的一半以上。墨西哥廠亦有擴增計畫。

## 核心競爭優勢
1.  **低軌衛星領域領導地位：** 作為SpaceX（星鏈）用戶端產品（包含用戶終端設備、天線及路由器）的主要供應商，且被視為地面接收站體的獨家供應商，在快速成長的低軌衛星市場中具備顯著優勢和高市佔率。
2.  **高階網通技術布局：** 積極投入企業用800G高階交換器、AI網路基礎建設所需的光學交換器（OCS）及Wi-Fi 7解決方案，這些高附加價值產品有助於提升毛利率並搶佔AI基礎設施升級商機。
3.  **多元且廣泛的產品組合：** 產品線橫跨網通、車用電子、智慧家庭，涵蓋廣泛的無線通訊應用，有助於分散風險並抓住多個產業趨勢的成長機會。
4.  **供應鏈韌性與產能擴張：** 透過積極擴充越南與墨西哥製造基地，優化全球產能佈局，降低地緣政治風險，並能滿足客戶對於供應鏈彈性的需求。
5.  **研發創新實力：** 在Wi-Fi 7、5G、邊緣AI應用、後量子密碼（PQC）模組及獨特的節能光學交換器等領域持續投入研發，確保技術領先地位。

## 法說會重點
**最近一次法說會日期：** 未找到 2025-2026 年的最新正式法說會確切日期，目前報導多為公司自結財報公布後的媒體溝通。

**管理層對各產品線的具體 guidance：**
*   **2026年營收展望：** 董事長謝宏波表示，以訂單能見度觀察，2026年營收可望實現雙位數成長，並看好成長幅度有機會超越公司設定的100億元目標。公司目標在三年後（約2028年）全年營收挑戰50億美元（約新台幣1,500億元）。
*   **低軌衛星：** 法人預估，啟碁來自低軌衛星的營收貢獻超過一成，為全台低軌衛星營收貢獻最大的公司，2026年營收占比將進一步提升，預期將維持高雙位數成長。
*   **高階網通：** 外資看好Wi-Fi 7解決方案、800G交換器與低軌衛星（LEO）使用者終端將成為啟碁2026年三大成長引擎。美系交換器廠商的企業用800G高階交換器產品，可望在2026年第三季開始出貨，預估佔當年營收比重上看5%（約50至60億元）。
*   **光學交換器 (OCS)：** 啟碁的全光交換器OCS可望在2027年開始出貨給美系光通訊廠商，貢獻營收。
*   **車用產品：** 2026年車用產品預計持續穩健成長。
*   **AI應用：** 公司已在智慧家庭與工業領域同步布局AI應用。
*   **成本壓力：** 儘管面臨記憶體漲價挑戰，公司已採取多元方案與供應鏈協調分攤成本，且客戶對價格接受度具備一定水準。

**產能利用率、資本支出金額：**
*   僅提及越南廠區產能逐步放量，但未提供具體的產能利用率數字。
*   2026年資本支出預估落在**新台幣25～30億元**之間，將維持在高檔水準，主軸集中在越南新廠與宿舍建置，以及墨西哥廠擴充規劃。

## 產業分析

### 市場規模與 CAGR 成長率
啟碁業務橫跨多個高速成長產業，受惠於以下趨勢：
*   **全球網路基礎設施市場：** 預計市場規模將從2024年的**2,487.8億美元**成長至2026年的**2,857.3億美元**，至2035年將達**5,328.6億美元**，複合年增長率（CAGR）為**7.17%**。
*   **全球網路設備市場：** 預計將從2024年的**582.4億美元**增長到2033年的**1,110.9億美元**，期間年複合增長率（CAGR）為**7.44%**。
*   **衛星互聯網市場 (低軌衛星相關)：** 2025年市場規模為**124億美元**，預計2026年將達到**142億美元**，並在2026-2035年期間以**14.3%**的複合年增長率成長，到2035年將達到**474億美元**。全球LEO衛星通訊服務支出預計在2026年將達到**148億美元**，年增**24.5%**。
*   **車用半導體市場 (車用電子相關)：** 預計從2024年的**677億美元**穩健增長至2029年的近**969億美元**，2024-2029複合年增長率(CAGR)達**7.4%**。
