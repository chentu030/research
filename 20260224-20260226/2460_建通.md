# 2460 建通 深度研究報告

## 一句話摘要
建通 (2460) 正經歷全面轉型，從傳統端子製造商積極跨入高階特殊新型銅材領域，並聚焦AI散熱、新能源車及中國市場新國標商機。在三大生產基地明確分工與新產品效益發酵下，2026年營運有望迎來翻轉，力拚全年轉盈並達成雙位數營收成長。

## 公司概覽
建通是一家台灣電子零組件製造商，主要經營精密電子產品及其零件、電器類機器的製造、加工，以及銅的買賣、電鍍加工、煉銅和銅鑄造等業務。公司近年轉型有成，從傳統端子延伸至高階特殊新型銅材應用。

### 核心產品線與相關題材
*   **傳統端子製品：** 應用於電器電源線、車用電源線、排插及適配器等。
*   **特殊新型銅材：** 涵蓋高導電匯流銅排、異型導體銅材、平板材、細晶銅桿與無氧壓延銅箔等金屬材料，可做到無氧銅、低雜質，提供更好的導電率。主要應用於AI散熱/AI伺服器（Busbar、微通道液冷MLCP）、新能源車及半導體導線架等高階領域。
*   **新型中國標準半絕緣插頭：** 具有發明專利製程優勢，因應中國2027年起全面強制使用的新規範。

## 核心競爭優勢
1.  **高階特殊銅材技術領先：** 台灣廠專注高導電率、低雜質的無氧銅、細晶銅桿、異型材等特殊銅材開發與生產，切入AI散熱（高導電匯流銅排、MLCP）、新能源車及半導體導線架等高附加價值應用，形成技術護城河。
2.  **中國新國標專利優勢：** 獨家「一次成型」發明專利製程，使新型中國標準半絕緣插頭在製程效率與成本上具備高競爭力，可望在2027年新國標強制實施後搶佔主要市場份額。
3.  **新能源車供應鏈認證：** 異型導體銅材已通過全球最大車用連接器廠泰科（TE Connectivity）認證，顯示其產品品質與可靠性達到國際一線大廠標準，有助於擴大在電動車市場的滲透率。
4.  **彈性與成本控制能力：** 透過越南廠的自有煉銅製程與外購銅下料垂直整合，以及中國廠區的明確分工與產線調整（東莞廠停產），有效控制生產成本並提升營運效率。
5.  **環保法規趨勢受惠：** 在歐盟銅合金鉛含量規範預計於2026年底到期，以及全球節能趨勢下，建通在環保微無鉛銅製品及高導電銅材的技術優勢將使其受惠於市場的替代需求。

## 法說會重點 (2025年11月27日)

建通於法說會中表示，儘管2025年前三季營運承壓，但集團的轉型佈局已全面落地，三大生產基地分工明確，並逐步形成新的成長動能。特殊新型銅材、新半絕緣插頭與新能源車用異型導體銅材將是2026年營運的主要推力。

### 管理層對各產品線具體 guidance
*   **特殊新型銅材 (台灣廠)：**
    *   已切入AI伺服器散熱與電力應用Busbar等原料供應，並與客戶合作開發微通道液冷產品(MLCP)。
    *   2025年已貢獻營收約1.8億元，佔1-10月合併營收8.63%。
    *   2025年主要客戶採訂單式生產，預計2026年將與主要客戶洽談排定銷售年約。
    *   預期2026年產出與營收有機會成長50%到100%，成為驅動毛利回升的核心動能。
*   **新型國標半絕緣插頭 (中國蘇州廠)：**
    *   鎖定GB1002-2024版強制規範商機，該規範將於2027年8月起全面使用。
    *   建通憑藉「一次成型」發明專利製程具備高競爭力，目前已獲各大電器廠商認證並啟動採購轉換。
    *   2025年第二季起營收逐步增加，目前相關產品營收占比約25%，預期2026年下半年有機會明顯放量，營收比重目標拉升至約75%。預計自2026年將迎來一波強勁的更換潮訂單。
*   **異型導體銅材 (蘇州廠)：**
    *   受惠於新能源車市場，2025年前10個月營收已達2024全年度的1.5倍。
    *   產線已達16條，預計2026年再增加4條。
    *   已獲全球最大車用連接器廠TE Connectivity (TE) 認證通過。
*   **傳統端子 (越南廠)：**
    *   採取垂直整合策略，維持自有煉銅產線高檔運轉，並增加外購銅下料，以發揮成本與產能優勢。2025年度越南廠煉銅產量將維持在高檔水準。
*   **整體營運與訂單能見度：**
    *   整體訂單能見度約一個月，市場普遍採取急單模式。
    *   建通對2026年營收達成雙位數成長抱持樂觀態度，並以2026年整體營運轉盈為目標。

### 產業趨勢

1.  **AI發展帶動高速、高密度與高效率互連技術革新：**
    *   **具體影響：** AI算力集中化使得資料中心內部互連密度與頻寬需求提升，**800G**傳輸逐步主流，**1.6T**進入設計驗證。推動高速互連技術（如支援**224Gbps PAM-4速率**的高速背板與板對板解決方案，以及**400/800Gbps聚合速率**的高速可插拔I/O連接器）。AI伺服器功率密度上升，促使液冷散熱（包括直接晶片冷卻、浸沒式冷卻）大規模滲透（**2026年AI晶片液冷滲透率達47%**）。**48V架構**日益普及。這些趨勢對連接器和銅材提出了更高導電率、更低損耗、更高效散熱、更高可靠性的要求。
2.  **電動車普及驅動高電壓、高電流連接器與端子需求：**
    *   **具體影響：** 全球電動車（EV）轉型加速，EV的高電壓、高電流需求，使得汽車用端子和連接器必須具備更高的耐久性和性能。推動對高壓連接器、充電插頭、高安全高效電力傳輸連接器需求。
3.  **綠色合規與材料轉換的壓力：**
    *   **具體影響：** 環保法規持續影響材料選用。例如，歐盟RoHS指令中關於銅合金鉛含量的過渡條款預計於**2026年底**到期，未來無鉛或低鉛材料應用比例將提高，引發材料替換、產品設計調整與供應鏈轉單。

### 對 建通 而言的具體機會和威脅
*   **機會：**
    *   **AI散熱商機：** 高階特殊新型銅材 (無氧銅、高導電匯流銅排) 直接受惠AI伺服器對高導電匯流銅排及液冷散熱材料的強勁需求，與客戶合作開發微通道液冷產品（MLCP）。
    *   **新能源車市場成長：** 異型導體銅材與車用連接器解決方案符合電動車產業高電壓、高電流需求，通過全球最大車用連接器廠TE Connectivity認證。
    *   **中國新規範商機：** 具發明專利製程優勢的新型中國標準半絕緣插頭，可望因應**2027年8月**起強制使用的新規範，搶佔市場先機。
    *   **環保微無鉛銅製品優勢：** 面對歐盟RoHS等環保法規對無鉛材料需求，具備競爭優勢，有望在合規市場獲得更多訂單。
*   **威脅：**
    *   **原材料價格波動：** 銅價飆升及貴金屬成本上漲可能壓縮毛利率。
    *   **技術快速迭代：** AI、電動車等新興產業技術發展迅速，若技術創新或產能擴充不及時，可能面臨競爭劣勢。
    *   **市場競爭加劇：** 高端連接器和特殊銅材市場吸引眾多廠商投入，競爭激烈。

### 相關投資題材的具體連結
*   **AI（人工智慧）：** 建通的高階特殊新型銅材，特別是「高導電匯流銅排」和液冷散熱解決方案，是AI伺服器高效能運算和優異散熱的關鍵零組件。隨著AI資料中心對運算力、互連技術與液冷散熱的需求成長，建通將直接受益於AI基礎設施的投資。
*   **電動車（EV）：** 建通積極轉型新能源車領域，提供車用連接器和高階銅材。電動車市場的快速成長，特別是對高壓、大電流、輕量化高導電銅材的需求，為建通帶來明確成長動能。
*   **HBM（高頻寬記憶體）：** 雖然建通非HBM製造商，但HBM作為AI晶片核心元件產生大量熱量，需要先進散熱方案。建通的「高導電匯流銅排」等高階銅材應用於AI伺服器，尤其在支援HBM等高功耗元件的散熱與電源傳輸方面，存在間接連結與商機。
