# 3515 華擎 深度研究報告

## 一句話摘要
華擎（3515）受惠於AI伺服器與工業電腦（IPC）兩大引擎強勁成長，2025年營收與獲利顯著提升，2026年隨NVIDIA B300伺服器放量出貨及IPC大單落地，營運展望樂觀，訂單能見度已達下半年，惟產品組合變化導致毛利率面臨壓力及記憶體成本上漲為潛在風險。

## 公司概覽
華擎科技（3515）成立於2002年，從華碩分割而出，是全球領先的電腦硬體製造商。公司核心業務已從傳統主機板逐步多元化，產品線涵蓋主機板、顯示卡、電競顯示器、電源供應器、迷你電腦、工業電腦及伺服器（包含AI伺服器）。旗下擁有「華擎」（消費性產品）、「永擎電子」（ASRock Rack，專注伺服器）及「東擎科技」（ASRock Industrial，專注工業電腦）等品牌子公司。

華擎近年積極轉型，重心轉向高成長的AI伺服器與工業電腦領域。

## 核心競爭優勢
1.  **AI伺服器領域的快速崛起**：旗下永擎電子專注於伺服器業務，已成功切入歐系客戶及多家美國、日本與歐洲的二線CSP業者供應鏈。公司積極佈局NVIDIA H100、B200、B300及AMD MI300等新世代AI伺服器，並於2026年第一季將B300產品放量，展現其在AI硬體設計與整合上的技術實力。
2.  **工業電腦業務的觸底反彈**：子公司東擎科技深耕工控與商用領域，受惠於北美串流服務商大單挹注，市場需求逐步回溫。其在邊緣AI應用上的佈局（如NVIDIA Jetson AGX Orin模組平台）將為未來成長注入新動能。
3.  **產品線多元化與高階化轉型**：除AI伺服器和IPC外，華擎在DIY市場致力於高階主機板及顯示卡發展，受益於AMD高階產品市佔提升，有效提高主機板平均售價（ASP）。多元化的產品組合分散了單一業務的風險。
4.  **靈活的生產策略與全球佈局**：華擎不設置自有工廠，採全面委外生產，具備彈性成本結構。生產據點分散於台灣、中國與越南，並預先完成美國組裝備案，以因應地緣政治與貿易壁壘，確保供應鏈韌性。

## 法說會重點
**最近一次法說會日期：** 2026年03月06日（富邦證券2026 Q1富邦企業日）
_註：本次法說會於本日舉行，其詳細內容與管理層給出的最新指引，目前無法從已發布的公開資訊中取得實時更新。以下重點內容綜合整理自2026年2月26日、2025年12月16日及2026年3月6日法說會前的相關報導。_

*   **管理層對各產品線的具體出貨量/訂單能見度說明：**
    *   **AI伺服器：** 總經理許隆倫指出，已獲得歐系客戶訂單，並切入多個二線CSP業者。2025年出貨輝達B200產品後，**2026年第一季B300產品可望放量出貨，成為主要成長動能，訂單能見度已達2026年第三季**，並將持續推進至下世代產品（如B400/B500）。子公司永擎約八成營收來自AI伺服器。
    *   **通用型伺服器：** 出貨同步成長，預計維持穩健的**雙位數年增長**，毛利率與平均銷售單價 (ASP) 穩定。
    *   **主機板：** 隨著市場殺價競爭緩解，ASP及獲利回歸正常，預期2026年第一季將有符合預期的表現。全年板卡產品力求維持與去年相當的水準。
    *   **顯示卡 (VGA)：** 市場需求熱絡，受惠於AMD顯卡動能，預期動能持續，有望呈現逐季成長。
    *   **工業電腦 (IPC)：** 旗下東擎受惠於歐美Cloud gaming客戶大單挹注及多個大型專案落地，預期**2026年將有更顯著的成長，上半年表現強勁**。
    *   **新產品 (電源供應器 / 顯示器)：** 電源供應器業務表現漸入佳境，預期2026年將持續推進。
*   **產能利用率、資本支出金額：** 未找到2024年以後最新的具體產能利用率資料。子公司永擎2025與2026年資本支出合計約**1至2億元**，主要用於研發、測試設備、量測能力與水冷實驗室。
*   **管理層給出的下季/下半年 guidance (2026年)：**
    *   **2026年第一季：** B300伺服器逐步成為主流產品，出貨量預計超越B200。
    *   **2026年上半年：** 子公司東擎將受惠於歐美Cloud gaming客戶大單陸續出貨挹注，提升出貨動能。
    *   **2026年下半年：** 伺服器業務表現有望比上半年更強勁，AI伺服器能見度已達下半年。
    *   **2026年全年：** 華擎持續看好AI伺服器與通用型伺服器業務帶來的成長驅動力，仍將扮演重要角色。法人預估年度稅後純益有望較去年成長三成，EPS將落在**19.77~23.58元**之間。

**利潤率變化原因分析：**
華擎利潤率自2024年高點逐漸下滑，主要原因為：
1.  **產品組合轉變：** AI伺服器業務營收比重持續上升，然其毛利率相對低於傳統主機板業務，導致合併毛利率承壓。公司策略是以量補價，以維持整體營運成長。
2.  **成本壓力：** 記憶體價格在2025年下半年漲幅達30%至50%，且預期漲勢將延續至2026年上半年，侵蝕主機板與顯示卡業務的毛利率。華擎將動態調整產品報價以反映成本。
3.  **工業電腦復甦：** 雖然工業電腦業務毛利率較高，但目前佔比僅4%（2025Q3），其對整體毛利率的拉升效果仍有限。

**供需狀況：**
*   **AI伺服器：** 需求持續強勁，尤其北美CSP業者積極投資。然而，GPU、HBM、NAND Flash等關鍵零組件供應鏈緊繃，導致價格上漲及交期延長。伺服器CPU供應也全面收緊。
*   **主機板：** 記憶體（特別是DDR5）價格飆升，對PC和工業電腦產業造成成本壓力，可能影響終端買氣。
*   **工業電腦：** 市場在經歷2023-2024年去庫存後於2025年復甦。但記憶體缺貨與漲價對IPC產業造成雙重打擊，即便訂單在手也可能無法如期交貨。

**產業的平均毛利率水準：**
*   **AI伺服器相關：** 具備高單價與高毛利特質。例如，伺服器導軌川湖毛利率76.05% (2025E)，散熱奇鋐水冷伺服器毛利率28-29% (2026E H1)，台達電AI產品組合優化毛利率34.3% (2025E)。
*   **工業電腦：** 研華預估2026年Q1毛利率38-40%。
*   **主機板/通用伺服器：** 未找到2025-2026年具體平均毛利率數據，但一般而言，通用型伺服器與主機板的毛利率相較AI伺服器更為競爭。

**供需狀況：**
*   **AI伺服器：** 需求持續強勁，尤其北美雲端服務供應商（CSP）積極投資AI基礎設施。然而，GPU、HBM (高頻寬記憶體) 和NAND Flash等關鍵零組件供應鏈極度緊繃，導致相關企業級SSD需求爆發式成長，推升價格漲勢。伺服器CPU供應也全面收緊，AMD EPYC處理器交貨週期延長至8-16週，Intel部分Xeon產品交貨週期甚至長達6個月，且價格上漲。

*   **工業電腦：** 市場在歷經2023-2024年去庫存後，2025年可望迎來明顯復甦。然而，記憶體缺貨與漲價對IPC產業造成雙重打擊，即便訂單在手，也可能因關鍵零組件不足而無法如期交貨。擁有大型集團支撐的IPC業者在料源取得上相對佔優。

**產業的平均毛利率水準：**
*   **AI伺服器相關：** AI伺服器業務具備高單價與高毛利特質。伺服器導軌製造商川湖科技在2025年毛利率達76.05%。散熱大廠奇鋐受惠於高單價、高毛利的水冷伺服器產品比重提升，預期2026年上半年毛利率將顯著跳升至28%至29%。台達電受惠AI產品組合優化，2025年毛利率攀升至34.3%。
