# 1303 南亞 深度研究報告

## 一句話摘要
南亞（1303）成功轉型為AI電子材料與記憶體投資雙引擎，擺脫傳統石化低谷。在AI伺服器、HPC與記憶體需求強勁帶動下，電子材料營收比重顯著提升，轉投資南亞科（2408）獲利挹注豐厚，推動公司營運重返成長軌道，儘管傳統石化業務仍面臨供過於求壓力，但高值化策略與擴產計畫為其帶來結構性成長動能。

## 公司概覽
南亞（1303）是台灣塑化產業的龍頭之一，業務多元，涵蓋塑膠加工、化工產品、聚酯產品及電子材料等四大領域。近年來，公司積極調整產品組合，深化高值化與差異化布局，電子材料事業在AI與高效能運算趨勢下，成為主要成長動能。

**主要產品線與應用：**
*   **電子材料：** 銅箔基板 (CCL)、銅箔、環氧樹脂 (Epoxy)、玻纖布、載板材料等，廣泛應用於印刷電路板 (PCB)、半導體、AI伺服器、高速網通、資料中心、智慧手機。
*   **塑膠加工：** PVC軟硬質膠布、管材、異型材等，應用於建築、包裝、消費品。
*   **化工產品：** 乙二醇 (EG)、雙酚A (BPA)、醋酸乙烯 (VAM) 等，為合成纖維、塑膠、塗料等產業重要原料。
*   **聚酯產品：** 聚酯粒、聚酯纖維等，應用於紡織、瓶用容器。

## 核心競爭優勢
1.  **高階電子材料領先地位：** 南亞在AI伺服器、高速運算所需的高頻高速電子樹脂、高階玻纖布 (Low CTE、Low DK)、高階銅箔及銅箔基板 (CCL) 領域具備領先技術與產能。特別是高階CCL材料，市場供不應求情況預計延續至2026年，賦予公司強大議價能力。
2.  **垂直整合能力：** 從基礎石化原料到終端加工產品的垂直整合，尤其在電子材料領域，從玻纖布、環氧樹脂、銅箔到CCL及載板材料，形成完整的供應鏈，確保關鍵原料供應穩定並提升成本效益。
3.  **轉投資南亞科（2408）的強勁挹注：** 南亞持有南亞科技約28.37%股權，在記憶體產業景氣復甦、DRAM合約價上漲及AI對HBM/DDR5需求帶動下，南亞科獲利將顯著貢獻南亞的業外收益，成為重要的獲利引擎。
4.  **積極的高值化與差異化策略：** 透過持續的研發投入，聚焦於電子材料、半導體及AI相關應用，擴大高階產品銷售比重，降低景氣循環對傳統石化業務的影響。

## 法說會重點
**最近一次法說會日期：** 2026年3月3日 (元大證券舉辦)。
**管理層發言與 guidance：**
*   **2026年Q1展望：** 電子材料需求如預期般強勁，農曆過年期間產線不停機。部分2月份營收將遞延至3月，預估整體第一季營收將優於2025年第四季。
*   **2026年全年展望：** 全年接單狀況顯示需求旺盛，管理層持審慎樂觀態度。隨著電子材料、塑膠產品需求緩步回升，加上AI應用帶動的電子級玻璃纖維布與電路板材料需求，本業營運轉趨穩健。
*   **電子材料事業：** 佔營收比重持續往上，2025年全年達46%，2025年第四季達51%，2026年1月至2月累計佔比更達到53%，公司看好後續佔比還有機會再提升。
*   **產品價格趨勢：** 電子材料事業旗下產品價格整體漲幅已超過雙位數。其中玻纖布漲幅最高，銅箔也可達到雙位數，CCL及環氧樹脂漲幅則為中高個位數。
*   **高階材料供需：** 高階CCL材料短缺情況預期難以在短期內緩解，供不應求狀況將延續至2026年。
*   **資本支出：** 南亞宣布，未來幾年海內外投資項目完工後，預計每年將可貢獻185.5億元的產值。其中，針對中國大陸的電子材料產線與美國軟質膠布擴產計畫，合計投資金額達137.9億元，預計於2026年至2028年間陸續完工。
*   **轉投資南亞科技 (2408)：** 南亞科2026年資本支出預計約500億元，較2025年實際支出134億元大幅成長273.13%，主要用於新廠廠務無塵室建置與設備採購。
