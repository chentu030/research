# 3062 建漢 深度研究報告

## 一句話摘要
建漢(3062)正值關鍵轉型期，受惠於越南新廠產能擴充、策略性轉向WiFi 7、企業級網通設備、光纖PON及低軌衛星（LEO）地面接收設備等高成長市場，並積極導入AI應用，預期2026年營收可望實現雙位數成長，本業也有機會轉虧為盈。儘管目前仍面臨產業競爭與供應鏈挑戰，但其前瞻性佈局及鴻海集團資源，將是推動其長期價值重估的關鍵。

## 公司概覽
建漢科技(3062)主要從事網通產品的研發、製造與銷售，並提供專業代工（ODM/JDM）服務，客戶涵蓋全球電信營運商與網通品牌。

**核心產品與服務**：
*   **寬頻與無線路由器**：涵蓋WiFi 6、WiFi 7無線基地台、WiFi Mesh路由器等，並積極從WiFi 6/6E升級至WiFi 7技術。
*   **網路交換器**：提供網管型交換機、第三/四層交換器等產品，主要針對企業端市場。
*   **光纖設備**：如電信PON（Passive Optical Network）產品。
*   **網路控制器**：包含雲端與地端網路控制器。
*   **其他網通設備**：如虛擬私有網路（VPN）、防火牆、Cable Modem、機上盒（STB）及網路攝影機。
*   **新興技術產品**：積極投入AI相關產品研發（如AI強化IP Camera）及低軌衛星（LEO）地面接收設備。

**營收結構 (2025年12月法說會資訊)**：

| :------------------------- | :--------- | :------- |

| 企業端產品                 | 32%        | 22%      |
| 電信光纖、機上盒（STB）及AI產品 | 31%        | -        |
| 其他                       | 4%         | -        |

**製造基地**：已全面轉移至越南，越南新廠於2025年全面投產，產能為既有產能的四倍。營收貢獻幾乎來自越南基地。

## 核心競爭優勢
1.  **非中製造與供應鏈韌性**：越南新廠於2025年全面投產，產能達既有四倍，所有產品線皆移往越南生產，有效降低生產成本並符合歐美客戶「China + 1」供應鏈重組要求，有利爭取國際訂單。
2.  **WiFi 7技術領先佈局**：積極從WiFi 6/6E升級至WiFi 7技術，並預期2026年WiFi 7產品出貨占比將提升至六成，具備搶佔新一代無線網路市場的技術優勢。
3.  **鴻海集團資源整合**：作為鴻海集團旗下的無線通訊設備製造商，享有集團在技術、客戶及供應鏈上的協同效應與資源支持。
4.  **產品結構優化**：成功將重心轉向更具潛力的企業端、電信光纖、AI及低軌衛星產品，降低消費性產品比重，提升產品組合的毛利潛力與成長性。
5.  **低軌衛星（LEO）先驅者**：作為鴻海集團低軌衛星佈局中的核心成員，負責研發與製造地面接收設備，切入高成長的新興通訊市場。

## 法說會重點 (2025年12月5日)
*   **產品發展策略**：將全力搶進WiFi 7、光纖及交換器新產品，企業客戶持續成長。產品發展策略從「點」轉變為「面」，目標是透過一個平台讓更多產品串聯，形成整體解決方案（Total Solution），推廣給潛在客戶。
*   **產品結構調整**：消費性產品比重從約40%降至**33%**；企業端交換器及無線路由器佔比從22%提升到**32%**；電信光纖、STB、AI產品佔比約**31%**。預期2026年三大產品業務營收占比將趨於均衡，各約佔3至4成。
*   **WiFi 7佈局**：客戶新開案以WiFi 7為主，並朝中高階產品發展，預期**WiFi 7出貨佔比將提升至六成**。
*   **新市場開拓**：將與客戶合作研究智慧家庭相關產品，進一步開拓美國市場。
*   **產能與資本支出**：越南新廠已投產，產能為既有產能的**四倍**。2025年第一季因越南新廠落成，需添置人力與設備，導致營業費用率上升。未提供具體的產能利用率或明確資本支出金額。
*   **2026年營運展望**：展望2026年，建漢預期營收可望有**二位數成長**，**本業有機會轉虧為盈**。無線網路、電信PON產品及交換器產品將是主要成長重點。
*   **訂單能見度**：因對等關稅影響，2025年第四季訂單已趨保守，對2026年上半年沒有確切預期，但2026年下半年營收較有機會轉好。
