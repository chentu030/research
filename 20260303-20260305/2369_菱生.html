2369 菱生 深度研究報告

公司概覽

菱生精密工業股份有限公司（股票代號：2369）是一家台灣的IC封裝與測試廠商，主要提供半導體零組件的封裝加工、測試、製造及買賣服務。

以下是關於菱生（2369）的詳細資訊：

1. 公司主要做什麼？核心產品/服務是什麼？

菱生的主要業務是積體電路（IC）的封裝與測試服務，以及發光二極體顯示燈的相關業務。 其核心產品/服務包括：

  • 積體電路封裝與測試：這是其最主要的業務內容。
  • 主要封裝產品類型：包括Small Outline Package (SOP)、Shrink Small Outline Package (SSOP)、Thin Small Outline Package (TSOP)、Thin Shrink Small Outline Package (TSSOP)、Plastic Leaded Chip Carrier (PLCC)、Quad Flat Package (QFP) 和 Thin Quad Flat Package (TQFP) 等。
  • 發光二極體顯示燈：也有此類產品的業務。
  • 重點布局產品：近年來，菱生積極布局高階封裝領域，特別是與AI應用相關的高速光元件封裝技術，以及扇出型封裝 (Fan-out) 等特殊應用。 此外，也強化在第三代半導體、車用產品以及Wi-Fi 7相關封裝的投入。
2. 各業務佔營收的大概比例

根據菱生2024年11月法說會以及2025年第三季的產品應用分析資料，其營收比重如下：

  • 感測元件 (Sensor IC)：約佔37% (2024年11月資料)，2025年第三季約佔32%。
  • 快閃記憶體 (Flash, 包含NOR Flash / NAND Flash)：約佔31% (2024年11月資料)，2025年第三季約佔37%，已成為營收佔比第一的業務，主要受AI應用擴展帶動。
  • 電源管理晶片 (Power IC / PMIC)：約佔20%。
  • 射頻元件 (RF IC)：約佔6% (2024年11月資料)，2025年第三季約佔5%。
  • 微控制器 (MCU)：約佔5%。
  • 其他 (Others)：約佔1%。
