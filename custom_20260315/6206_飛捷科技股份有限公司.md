# 6206 飛捷科技股份有限公司 深度研究報告

## 公司概覽

飛捷科技股份有限公司（6206）是一家工業電腦製造商，其業務正從單純的硬體生產轉型為提供軟硬體整合的智慧解決方案。

以下是關於飛捷科技的簡要介紹：

1.  **公司主要業務與核心產品/服務：**
    飛捷科技主要從事工業電腦 (IPC) 的設計與製造，核心產品包含端點銷售系統 (POS System)、平板電腦 (Panel PC)、自助服務系統 (KIOSK) 以及嵌入式電腦 (Embedded Box PC)。公司正積極從傳統硬體製造商轉型，結合人工智慧 (AI) 與雲端技術，提供「硬體＋軟體」的整合性解決方案。其產品線擴展至邊緣AI箱式PC (Edge AI Box PC) 和智能支付模組等。此外，飛捷旗下擁有三家軟體子公司：華捷智能 (Berry AI)、英諾菲 (Inefi) 和英捷智能 (Angible)，分別專注於智慧餐飲的AI影像辨識、跨裝置遠端管理平台以及零售自助結帳的AI視覺辨識技術，以期提供客戶「一次性硬體銷售 + 5~7 年軟體服務」的訂閱制模式。

2.  **各業務佔營收的大概比例：**
    根據2025年3月的資訊，產業電腦業務約佔飛捷營收的86%，週邊產品佔13%。然而，該資料來源也註明「尚未有最新資料」⚠️過時。
    值得注意的是，飛捷的軟體子公司在2026年預計將貢獻整體營收的5%，其中華捷智能 (Berry AI) 的營收有機會呈現倍數成長。

3.  **製造基地與各基地的營收貢獻比例：**
    飛捷科技的產品設計與製造主要在台灣進行，強調in-house（內部）自行設計、製造與測試，而非外包給代工廠，這有助於其研發整合與客製化解決方案的發展。
    目前公開資料中，未找到各製造基地及其營收貢獻的具體比例資訊。

4.  **所屬的細分產業和相關題材：**
    飛捷科技所屬的細分產業為工業電腦業 (IPC)，並積極拓展至智慧零售、智慧餐飲、智慧物聯網 (AIoT) 和邊緣AI (Edge AI) 等領域。相關題材包括全球「非現金交易需求」、「行動商務服務」以及「AI軟硬體應用需求」等，這些被視為推動未來終端應用換機潮的主要驅動力。

5.  **最近的重大消息 (2024年以後的資料)：**
    *   **2025年營運表現：**
        *   2025年全年合併營收達新台幣50.32億元，年增9.3%。
        *   2025年第四季營收為新台幣10.84億元。全年營業利益創歷史新高，儘管受業外匯損影響，稅後淨利略微年減，每股盈餘 (EPS) 達6.93元。

        *   2025年營運面臨新台幣大幅升值和上游供應鏈成本上揚導致的缺料問題，對營收與獲利造成壓力，預計此影響將延續至2026年上半年。

## 產業數據

### 該產業全球市場規模和 CAGR 成長率

工業電腦 (IPC) 市場在2025年預計為60.6億美元，並預計在2026年成長至63.6億美元。預測到2035年，市場規模將達到103.5億美元，2026年至2035年間的複合年成長率（CAGR）約為5.5%。 另一份報告預估，全球工業用電腦市場在2025年至2030年間將成長22.692億美元，預測期間的複合年成長率（CAGR）為6.8%。 還有資料顯示，工業用電腦市場預計將從2025年的69.16億美元成長到2030年的88.27億美元，複合年成長率為5.00%。

就POS系統而言，全球POS市場規模在近年來呈現飛躍式成長，預計將從2025年的538.4億美元成長到2026年的646.1億美元，複合年成長率（CAGR）達20.0%。預計到2030年將達到1,325.7億美元，CAGR為19.7%。 零售POS終端市場規模在2025年為296.9億美元，預計2026年將成長至333.5億美元，CAGR為12.3%。預計到2030年將達到518.4億美元，CAGR為11.7%。 商業POS系統市場規模預計在2025年為140.5億美元，2026年達到162.6億美元，2026年至2035年的CAGR為15.6%，預計到2035年將成長至599.1億美元。 POS軟體市場在2025年預計將成長至144億美元，CAGR為11.38%，預計到2032年將達到307.7億美元。

自助服務機 (KIOSK) 市場規模在近年來快速擴大，預計將從2025年的282.7億美元成長到2026年的311.4億美元，CAGR為10.2%。 零售自助服務亭市場規模在2025年為255.9億美元，預計2026年將成長至279.6億美元，CAGR為9.3%。 另一份報告則預計全球自助服務亭市場規模在2025年超過146.3億美元，預計到2026年將達到160.1億美元，2026年至2035年的複合年成長率將超過10.5%，預計到2035年將超過397.1億美元。

### 供需狀況：目前是供過於求還是供不應求？具體數據

工業電腦（IPC）市場在歷經2023-2024年的去庫存化後，2025年可望迎來明顯復甦。 2025年上半全球工業電腦（IPC）市場明顯復甦，台系IPC業者2025年上半營收規模年成長率達13.6%，顯示客戶庫存水位回歸健康、需求逐步回升，呈現產業「全面回溫」格局。 然而，記憶體缺貨問題推升價格攀升，特別是DDR5產品在短時間內出現倍數漲幅，已衝擊下游電子產業，導致PC與工業電腦（IPC）業者普遍面臨營運壓力，甚至出現「貨源短缺比漲價更棘手」的情況。 飛捷也表示，2025年營運環境面臨匯率與成本端雙重逆風，特別是下半年上游供應鏈成本上揚，引發產業結構性的缺料問題，相關影響預期延續至2026年上半年。

## 產業趨勢

### 影響該產業的 2-3 個關鍵技術趨勢和具體影響

1.  **邊緣AI (Edge AI) 與AIoT (人工智慧物聯網)：**
    *   **具體影響：** 隨著AI應用從雲端拓展至邊緣端，AI控制系統、機器視覺、地端大型語言模型（LLM）應用成為需求焦點。這帶動了工業電腦業者對邊緣AI設備的需求，促使業者積極結盟系統整合商（SI）與AI軟體夥伴，強化整合型解決方案能力。 工業電腦遇上AI、邊緣運算、自動化，將開啟更高附加價值與更廣泛的商機，例如智慧城市、智慧交通、智慧醫療、智慧工廠、AIoT、國防航太等。 資策會MIC預估，2026年邊緣AI硬體的滲透率將朝向2成邁進。 研華預計2025-2026年邊緣AI營收占比將提升至15%至20%。
2.  **軟硬體整合與訂閱制服務模式：**
    *   **具體影響：** 工業電腦產業正從單純的硬體銷售轉向提供「硬體+軟體」的整合性解決方案，並推動訂閱制服務模式。這能為客戶提供更完整的解決方案，並為廠商帶來持續性的軟體服務收入，提升毛利率。 飛捷的策略便是透過軟體子公司提供5~7年的軟體服務，透過AI影像分析等技術，賺取高毛利的SaaS軟體訂閱財。
3.  **非現金交易、行動商務服務與自助服務系統普及：**
    *   **具體影響：** 全球非現金交易需求、行動商務服務以及AI軟硬體應用需求的持續成長，將成為POS系統、KIOSK（自助服務系統）及Panel PC等終端應用換機需求的主要驅動力。 非接觸型支付的普及是POS市場成長的重要因素，因其提供了便利性、速度和安全性。 自助服務亭的普及也受非接觸式介面和先進支付處理技術的整合所推動。

### 對 飛捷科技股份有限公司 而言的具體機會和威脅

**機會：**

*   **邊緣AI應用擴大：** 飛捷在邊緣AI箱式PC的需求成長，以及其AI子公司華捷智能、英捷智能在智慧餐飲與零售AI視覺辨識的布局，將直接受惠於邊緣AI的爆發式成長。 華捷智能的AI影像分析方案已成功打入美國速食連鎖，顯示其AI軟體服務的潛力。
*   **軟體服務模式帶動高毛利：** 飛捷轉型為「硬體＋軟體」整合性解決方案提供商，並推行訂閱制模式，預計能顯著提升公司整體毛利率和獲利能力。軟體持續性收入規模的墊高，預計在2027至2028年將轉化為穩健的獲利動能。
*   **全球市場需求回溫：** 2025年工業電腦市場從去庫存進入復甦，半導體擴廠與邊緣AI導入帶動台系IPC業者營收成長。飛捷在歐洲和美國市場的強勁表現，顯示其抓住了市場回溫的機會。
*   **支付模組新產品與新應用場景：** 飛捷推出以「AI × Payment × Retail」為主軸的支付模組新產品，透過產品線擴張和加速新市場切入，有望拓展新客戶與新應用場景。

**威脅：**

*   **零組件缺貨與成本上漲：** 記憶體（DRAM、CPU等）價格飆升及供應鏈結構性缺料問題，預計將持續影響至2026年上半年，對營收與獲利構成壓力。 飛捷雖已採取動態調價和備料策略，但仍需面對成本波動風險。

*   **市場競爭加劇：** 儘管飛捷積極轉型，但工業電腦市場競爭激烈，主要競爭對手如研華也在積極布局邊緣AI與軟體訂閱制，這要求飛捷持續創新以維持競爭優勢。
*   **經濟不確定性：** 全球經濟景氣、地緣政治（如中東戰爭）等宏觀因素仍可能帶來不確定性，影響終端市場需求。
