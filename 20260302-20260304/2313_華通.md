# 2313 華通 深度研究報告

**今天日期：2026年03月06日**

## 一句話摘要

華通（2313）正積極轉型為全球高階PCB解決方案領導者，憑藉其在低軌衛星（LEO）領域的絕對優勢與AI伺服器/資料中心應用的深度佈局，搭配積極的海外擴廠計畫與產品組合優化，儘管短期面臨市場波動與法人籌碼調節，但預期2026年營運將迎來爆發性成長，毛利率與獲利能力有望顯著提升。

## 公司概覽

華通是全球高密度互連板（HDI）的領導廠商，核心產品涵蓋HDI板、多層板、軟板、軟硬結合板、IC載板與SMT組裝。公司已成功從傳統消費性電子PCB供應商轉型，積極擁抱太空科技與AI運算，成為相關關鍵基礎設施提供者。

**主要產品線與應用：**
*   **高密度互連板 (HDI)**：廣泛應用於行動電話、筆記型電腦、網通設備、伺服器、汽車電子、低軌衛星等。
*   **多層板 (MLB)**：用於高階伺服器、網通設備、資料中心、基地台等。
*   **軟板 (FPC) / 軟硬結合板 (Rigid-Flex)**：主要用於行動通訊、穿戴裝置等。
*   **IC載板 (Substrate)**：用於晶片封裝。
*   **SMT組裝 (Surface Mount Technology)**：提供客戶一站式解決方案。

**製造基地：**
華通在全球設有8座工廠，分佈於台灣、中國大陸（惠州、重慶）和泰國。
*   **台灣廠區**：維持高稼動率，新產能最快於2026年第二季開出，並將定位為高階產品研發與初期量產中心，專注開發AI伺服器相關產品。
*   **中國大陸廠區**：惠州、重慶設有產能。
*   **泰國廠區**：已於2024年投產硬板廠，員工逾1000人。產能逐步從簡單產品轉向高階衛星板，已擴至每月40萬平方呎，並預計在2026年將倍增至80萬平方呎，主力支援衛星應用。泰國廠二期則預計於2026年第四季放量，並同步啟動第三座工廠建置規劃，目標於2027年量產硬板產能。
*   各基地營收貢獻比例未提供2024-2026年的具體數據。

**所屬細分產業與相關題材：**
華通所屬產業為**印刷電路板 (PCB) 製造**，特別是**高密度互連板 (HDI)**。主要受惠題材包括：
1.  **低軌衛星 (LEO)**：全球低軌衛星板市占率高達80%，受益於SpaceX等運營商加速佈局及衛星規格提升。
2.  **AI伺服器與高階交換機**：切入AI伺服器CPU母板、配板供應鏈，加速佈局高階伺服器與光通訊應用。
3.  **AI智慧眼鏡**：受惠於Meta及小米等AI智慧眼鏡的需求。
4.  **高速光模組**：800G光模組已於2025年Q3末出貨，1.6T Switch HDI於2026年上半年推出，3.2T Switch HDI開發中。
5.  **高階HDI厚板**：20多層高階HDI厚板產品已建置完成，2026年下半年將出貨ASIC伺服器相關訂單。

## 核心競爭優勢

1.  **低軌衛星市場絕對領導地位**：華通在全球低軌衛星板市場擁有高達80%的市佔率，與SpaceX等主要客戶合作逾十年，在天上板供應佔比達9成，太空PCB板技術門檻極高。隨著低軌衛星部署加速及規格升級，此為華通獨一無二的護城河。
2.  **AI伺服器與高速運算關鍵供應商**：公司憑藉其深厚的HDI技術和高層數板製造能力，成功切入AI伺服器CPU母板、配板及高階交換機供應鏈，並積極佈局800G/1.6T光模組高階PCB板，為AI算力基建提供關鍵支援。
3.  **先進技術與研發能力**：華通在20多層高階HDI厚板、CoWoP等先進製程領域持續投入，展現其在高階板材製造的領先技術實力，能配合客戶開發最新的AI伺服器相關產品。
4.  **全球化彈性產能佈局**：在台灣、中國大陸及泰國設立生產基地，尤其泰國新廠的擴建，有效分散地緣政治風險，滿足客戶「非中非台」生產要求，並專注支援衛星與資料中心等高階應用，提升供應鏈韌性。
5.  **產品組合持續優化**：透過聚焦低軌衛星、AI伺服器等高附加價值產品，逐步降低消費性電子產品比重，有助於提高整體毛利率與獲利水平。
