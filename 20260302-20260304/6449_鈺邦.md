# 6449 鈺邦 深度研究報告

## 一句話摘要
鈺邦 (6449) 為全球固態電容領導廠商，受惠於AI伺服器、AI PC與電動車等高成長應用需求爆發，積極擴充V-Chip與CAP等高階產品產能，並透過泰國新廠分散風險。受惠於產品組合優化及產能擴張，法人預期其2026年營收及EPS將續創新高，展現強勁成長動能。

## 公司概覽
鈺邦科技（6449）主要從事固態電容的研發、設計、製造及銷售，是鋁質電解電容領域的被動元件製造商，產品廣泛應用於AI伺服器、GPU、PC、車用電子等高階市場。

**核心產品線：**
*   **捲繞型SMD (V-Chip)**：體積小、導電性能佳、可自動化貼片，應用於AI伺服器、GPU、高階主機板、筆記型電腦。
*   **晶片型固態電容 (CAP / SP-Cap)**：高階產品，應用於NVIDIA伺服器與顯示卡、筆記型電腦及AI伺服器。
*   **疊層式晶片固態電容 (SMLCC / SMLC)**：子公司細伎科技開發，目標取代鉭質電容，已送樣筆電、伺服器、電源、SSD客戶測試。
*   **混合型電容 (Hybrid)**：重點佈局車用電子市場，亦用於伺服器與高階電源。
*   **捲繞式固態電容 (DIP)**：主要應用於主機板與電源供應器。
*   液態電解電容 (EL-CAP)。

## 核心競爭優勢
1.  **AI伺服器/GPU核心供應商地位：** 鈺邦為NVIDIA V-Chip第一供應商（約佔8成），CAP第二供應商（約佔3成），深耕NVIDIA GB200/GB300系列伺服器及NVLink Switch等AI高階應用，受惠於AI晶片對高階固態電容數量與規格的大幅提升。
2.  **領先的技術與產品組合：** 專注於高導電性高分子材料電容器，具備15年經驗。其SMLC產品旨在取代鉭質電容，Hi-Rel CAP針對AI GPU模組開發，具備125°C、5,500小時耐受能力。Hybrid產品積極佈局車用電子市場。
3.  **穩定的國際大廠客戶群：** 服務HP、Lenovo、Acer、Asus、AMD、Meta、Nvidia及Amazon等一線品牌客戶，確保訂單能見度與成長動能。
4.  **積極的全球化產能擴充：** V-Chip、CAP等高階產品產能持續擴張，並建立泰國新廠作為非中生產基地，有效分散地緣政治風險，並提升供應鏈韌性。
5.  **日系廠商轉型帶來的市場機遇：** 日本傳統電容巨頭逐步退出中低毛利通用型市場，為鈺邦等在高階市場加速滲透和擴大市佔率提供了有利環境。

### 產業趨勢
1.  **高功率、高頻率、耐高溫、小型化、長壽命：** 5G、AI伺服器、電動車等應用推升對電容器在高頻低阻抗、耐受更高工作溫度（如135°C以上）、小型化及長壽命（如12萬小時）的要求，驅動先進材料與製程創新。
2.  **固態與混合型電容的發展及替代：** 固態電容和導電高分子固液混合型電容（Hybrid）因高穩定性、低漏電流、長壽命等優勢，逐步替代傳統液態電解電容。SMLC產品目標取代價格昂貴且具地緣政治風險的鉭質電容。
3.  **AI與先進封裝對電源完整性要求提升：** AI加速器晶片功耗與封裝面積增長，對電源完整性（Power Integrity）需求大增，高密度矽電容（如愛普*的S-SiCap）和整合式穩壓器（IVR）成為關鍵。

**對鈺邦的具體機會與威脅：**
*   **機會：**
    *   **AI伺服器與AI PC需求爆發：** 鈺邦的V-Chip和CAP產品在NVIDIA GB200/GB300系列中佔主導，AI PC換機潮帶來顯著成長。
    *   **車用電子市場擴張：** 混合型電容在車用電子市場應用持續增加。
    *   **取代鉭質電容：** SMLC產品搶佔鉭質電容替代市場。
    *   **日系廠商轉型：** 日本巨頭退出中低毛利市場，為鈺邦等在高階市場提供擴張機會。
    *   **非中產能擴充：** 泰國新廠降低地緣政治風險，滿足客戶對供應鏈彈性需求。
*   **威脅：**
