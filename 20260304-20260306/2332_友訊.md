# 2332 友訊 深度研究報告

**今天日期：2026年03月06日**

## 一句話摘要
友訊（D-Link, 2332）正從傳統網通硬體供應商加速轉型為「AI雲端」和整合型解決方案提供者，受惠於Wi-Fi 7換機潮、全球資安意識提升以及日、中東等市場大型專案訂單，營收動能於2025年第四季觸底回升，展望2026年，儘管全球經濟仍存不確定性，然公司產品組合優化及新訂單將帶動毛利率與獲利能力持續改善，為其帶來成長契機。

## 公司概覽
友訊科技（D-Link）成立於1986年，是一家全球領先的網通品牌，業務核心正從單純的硬體銷售轉向「AI雲端」和整合型解決方案。公司積極調整產品發展策略，以使用情境為核心，全面涵蓋個人行動、家庭應用、商用環境與垂直產業的網路需求。

### 業務與產品線
友訊科技主要從事企業與消費性電腦網路系統及設備、無線通訊產品及其零組件的研發、製造與銷售，並積極布局工業物聯網 (IIoT) 市場及電信營運商解決方案。

*   **交換器 (Switches)**：企業級網路交換器，如支援Wi-Fi 6E/7需求的10G PoE網管型交換器、2.5G PoE智慧型交換器，以及100G/25G L3核心交換器。2025年推出了DMS-108P、DMS-1016、DMS-1024等2.5G多埠非網管型交換器，以及DMS-1250系列2.5G智慧型網管交換器。亦提供工業用交換器及微資料中心交換器DQS-5000-56ZS。
*   **無線網路設備 (WLAN Products)**：涵蓋無線路由器與無線基地台（AP）。在Wi-Fi 7技術方面，2025年8月推出了M95 BE9500 Wi-Fi 7智慧Mesh路由器、R95 BE9500 Wi-Fi 7智慧路由器、G572 5G NR BE7200 Wi-Fi 7路由器。2025年12月發表企業級無線產品組合DAP-E9560 BE9500 Wi-Fi 7吸頂式無線基地台和DAP-X3060W AX3000 Wi-Fi 6壁掛式無線基地台。2026年2月推出DAP-E9560 BE11000 Wi-Fi 7三頻PoE企業級無線基地台。
*   **寬頻網路產品**：XGS-PON數據機、LTE/5G行動通訊寬頻裝置，以及5G CPE（客戶終端設備）產品。
*   **數位家庭應用**：網路儲存產品、智慧插座、整合AI功能的數位攝影機（IP Cam）和中央監控系統。
*   **雲端解決方案**：mydlink雲端解決方案、D-Link Nuclias網路管理平台（SOHO, Connect, Hyper），以及D-ECS雲端管理平台。
*   **行動工作解決方案**：旅行用路由器、多功能集線器、氮化鎵充電器與行動電源。

### 營收結構
根據2024年12月的資料顯示，友訊科技的營收有**70%以上**的產品直接銷售到終端消費者，其中**超過60%**銷售到新興市場。**目前未找到 2025-2026 年按具體產品線（如交換器、無線產品等）劃分的最新營收貢獻比例資料。**

### 製造基地
友訊科技採取「全球生產基地布局、多元供應鏈管理」策略，運用中國供應鏈的優勢，同時強化台灣的研發與企劃能量，以維持產品價格競爭力與技術自主性。公司強調其「台灣製造（MIT）」的品質承諾，其晶片、軟體、製造等台灣供應鏈佔友訊整體供應鏈的**65%**。**目前未找到 2025-2026 年各製造基地的具體營收貢獻比例的最新資料。**

## 核心競爭優勢
1.  **策略轉型與產品差異化**：友訊積極從單純硬體銷售轉向以「AI雲端」和「整合型解決方案」為核心，產品導入AI功能（如AQUILA PRO AI系列榮獲2025年紅點設計大獎和2024年中東Future Security Awards的「最佳AI創新產品獎」），提升產品附加價值和競爭力。
2.  **前瞻技術布局**：公司積極布局Wi-Fi 7、5G CPE等新一代通訊技術，推出多款家用與企業級路由器及AP，搶占新標準的換機潮，確保技術領先地位。
3.  **資安與品牌信譽**：資安產品獲得美國企業級客戶五年大訂單，且旗下Mesh路由器通過歐盟EN 18031資安認證，凸顯其產品高安全性，有助於在日益重視資安的市場中擴大市佔。
4.  **彈性供應鏈與MIT策略**：透過「全球生產基地布局、多元供應鏈管理」活用中國供應鏈優勢，同時強化台灣研發與65%的台灣製造供應鏈，平衡成本與品質，並受惠於地緣政治緊張下「去中化」的產業趨勢。
5.  **ESG永續發展**：重視產品碳足跡與永續設計，榮獲2024年商業周刊與台大風險中心合辦的「碳競爭力百強」通信網路業第一名，提升企業形象與品牌價值。

## 產業分析

### 市場規模與成長率
網通產業正受惠於5G普及、雲端數據中心擴展及智慧城市建設等多重驅動。

*   **全球網路基礎設施市場**：預計在2025年達到**2,666.1億美元**，2026年將進一步成長至**2,857.3億美元**，並預計在2024年至2035年間以**7.17%**的複合年增長率（CAGR）增長，至2035年有望激增至**5,328.6億美元**。
*   **全球網路設備市場**：預計從2024年的**582.4億美元**成長至2033年的**1,110.9億美元**，期間CAGR達**7.44%**。
*   **交換器市場 (Switches)**：全球乙太網路交換器市場（含企業級與資料中心）預計將以**8.09%**的CAGR成長，從2025年的**372.5億美元**增至2030年的**549.6億美元**。
*   **無線網路設備 (WLAN Products)**：
    *   全球Wi-Fi市場規模在2025年預計為**184.8億美元**，2026年將增至**210.6億美元**，並預計在2026年至2031年間以**13.94%**的CAGR成長，至2031年達到**404.4億美元**。
    *   家用Wi-Fi路由器市場預計從2025年的**57.6億美元**成長至2035年的**111.1億美元**，2026年至2035年的CAGR為**6.78%**。

### 供需狀況
目前網通產業整體呈現**需求強勁**的態勢，特別是高速、高容量設備方面。生成式AI、雲端計算與邊緣運算的快速發展，使得資料中心對頻寬的需求從Gbps級躍升至Tbps級，推動800G甚至1.6T交換器的需求。企業網路升級需求亦來自於Wi-Fi 7的普及與舊型交換器的更換週期。電信商需求持續回溫，特別是美國與多國積極推動的網通基礎建設，將帶動Wi-Fi 7、交換器與高速光纖設備出口回溫。雖然2025年部分客戶庫存去化曾導致營收小幅衰退，但到2025年第四季已重回成長軌道，整體市場對新一代高速、高容量網通設備的需求旺盛，尤其是在AI基礎設施投資的超級週期下。

網通產業的毛利率因產品類型、技術含量及競爭策略而異。
