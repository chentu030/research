# 6919 康霈* 深度研究報告

## 一句話摘要
康霈生技 (6919) 專注於開發新型醫美體雕與慢性發炎新藥 CBL-514，其獨特的誘導脂肪細胞凋亡機制，鎖定非手術局部減脂、GLP-1 併用體重管理及罕病竇根氏症等龐大市場。公司手握充裕現金達 **94 億元**（截至2025年Q2）支持未來研發，短期營運雖處虧損，但多項臨床試驗進度、高額全球授權金潛力及亞洲市場布局是未來主要成長動能。

## 公司概覽
康霈生技的核心業務是新藥研發，特別著重於醫美體雕與慢性發炎疾病領域。其核心產品為：
*   **CBL-514注射劑**：透過誘導脂肪細胞凋亡來精準減少注射部位脂肪的全新小分子新藥。適應症涵蓋非手術局部減脂、中/重度橘皮組織以及體重管理（改善復胖）。
*   **CBL-514D**：CBL-514 的同成分不同劑型新藥，適應症包含罕見疾病竇根氏症（Dercum's disease）。

康霈目前仍處於新藥開發階段，尚未有產品上市銷售，因此營收規模偏小，且部分月份營收為零。公司目前的營收主要來自健康食品銷售，但因非每月固定出貨，導致單月營收波動劇烈。

## 核心競爭優勢
1.  **獨特的作用機制**：CBL-514 透過「誘導脂肪細胞凋亡」的全新小分子機制，與現有市場上能量型設備或傳統溶脂產品形成差異化競爭優勢。
2.  **多重適應症布局**：涵蓋醫美局部減脂、中/重度橘皮組織、體重管理（GLP-1 併用改善復胖）及罕見疾病竇根氏症，市場潛力廣闊。
3.  **強勁的全球專利保護**：擁有 **188 件**專利申請，其中 **107 件**已獲核准，專利保護至 **2045 年**，提供長期競爭壁壘。
4.  **充裕的研發資金**：截至 2025 年第二季，手握現金及定存達 **94 億元**，足以支應未來三年預估約 **30 億元**的研發費用，確保臨床試驗順利推進。
5.  **經驗豐富的國際團隊**：積極擴充肥胖與代謝疾病國際顧問團隊，並與國際權威專家網絡合作，強化研發與市場拓展能力。

## 法說會重點
**最近一次法說會日期**：2026年3月3日 (元大證券第一季投資論壇)；2026年2月6日 (2026年第一季線上法人說明會)。

**管理層給出的 2026 年臨床與營運策略**：
*   **全球與區域臨床雙軌並進**：持續推進 CBL-514 減少皮下脂肪全球樞紐三期試驗 (SUPREME-01, SUPREME-02)、聯用 GLP-1RA 二期試驗 (CBL-0202EFP)，以及罕病竇根氏症 Phase 2b 等多項關鍵臨床試驗。
*   **亞洲臨床布局**：規劃亞洲市場臨床布局，包括在台灣、澳洲啟動 CBL-0206 Phase 2 試驗，以及在中國、加拿大啟動 CBL-0304 Phase 3 試驗，以拓展亞洲族群臨床數據並對接中國藥監局 (NMPA) 藥證申請。
*   **CBL-514 併用 GLP-1R (Tirzepatide) Phase 2** 已獲美國 FDA IND 核准。
*   **CBL-514D 竇根氏症 Phase 2b** 預計下半年取得最終統計數據。
*   **營運策略**：從醫美局部減脂拓展至肥胖與代謝領域，建立「全球+區域授權」的雙引擎獲利模式。
*   **研發時程表 (2026-2028年)**：
    *   **2026年上半年**：主要進行 SUPREME-01 (美、加) 受試者招募與 CBL-0202EFP Phase 2 (美) 量表開發及驗證。
    *   **2026年下半年**：多個臨床試驗的 IND 申請，包括 SUPREME-02 (美、加、澳) Phase 3、CBL-0303 Phase 3 Long Term FU (美)、CBL-0206 Phase 2 (台、澳) 及 CBL-0102 PK & CBL-0304 Phase 3 (中國臨床)。竇根氏症的 CBL-0202DD Phase 2b 試驗結果預計於下半年取得。

### 產業趨勢
1.  **非侵入性與微創治療的普及化與精緻化**：
    *   **趨勢**：消費者傾向於創傷小、恢復期短、風險低的非侵入性和微創美容手術，佔全球美容手術總量 **54% 以上**。
    *   **影響**：刺激更多非手術減脂、皮膚緊緻療程發展。CBL-514 作為注射劑符合此趨勢，若能提供高效、恢復期短的局部減脂方案，具競爭力。
2.  **AI 和數位健康平台的深度整合**：
    *   **趨勢**：AI 驅動的治療計劃、自動化設備、虛擬諮詢等正加速應用。醫美器械技術迭代週期縮短至 **18-24 個月**。
    *   **影響**：提升治療精準度、安全性、個性化。康霈未來可將 AI 應用於臨床數據分析、藥物篩選，甚至輔助醫生治療方案規劃。
3.  **膠原蛋白增生與再生醫學**：
    *   **趨勢**：市場對刺激自身膠原蛋白生成的療程需求增加，從「填補」轉向「重塑」。
    *   **影響**：提供更自然、持久的效果。這與 CBL-514 的減脂機制不同，但反映市場對深層、自然美容效果的追求。

**對康霈* 而言的具體機會和威脅**：
*   **機會**：非侵入性體雕市場快速成長、CBL-514 全新作用機制帶來差異化優勢、市場對個性化高效療程需求、慢性發炎疾病新藥開發潛力（竇根氏症）。
*   **威脅**：醫美市場競爭激烈、新藥研發失敗風險、高昂的法規遵循與上市成本、消費者信心與潛在副作用、醫美市場價格競爭。
