# 3694 海華 深度研究報告

## 一句話摘要
海華科技 (3694) 受惠於Wi-Fi 7/8技術升級、AIoT與機器人應用擴張，以及遊戲機、印表機和白色家電等多元產品線的出貨動能，2025年獲利表現亮眼，累計前三季EPS達3.27元，全年營收達114.07億元，展望2026年營收與獲利有望延續雙位數成長，然需留意原物料價格波動與總體經濟逆風。

## 公司概覽
海華科技主要從事無線通訊模組和數位影像模組的研發、生產與銷售，為和碩聯合科技股份有限公司的最終母公司，其產品廣泛應用於筆記型電腦、遊戲機、消費性電子、穿戴式裝置、車載電子、無人機、機器人及物聯網產品。

*   **核心產品**：
    *   **無線通訊模組**：涵蓋WiFi模組（如WiFi 7、WiFi 8）及5G模組等。
    *   **數位影像模組**：包括相機模組、3D鏡頭模組、IR紅外線鏡頭模組、ToF測距鏡頭模組，以及創新的eUSB2 V2鏡頭模組等。

*   **營收結構 (2024年)**：
| 業務類別         | 營收比重 |
| :--------------- | :------- |
| 無線通訊模組     | 83.6%    |
| 數位影像模組     | 15.9%    |
| 其他             | 0.5%     |

*   **製造基地**：主要位於中國大陸上海。

*   **所屬產業及相關題材**：
    *   **產業**：電子–通信網路產業、通訊設備零組件、網通設備。
    *   **題材**：WiFi 7/8、AI應用 (AI無線模組、AI相機模組、機器人AI視覺辨識、邊緣運算)、物聯網 (無人機、機器人、安防、智慧眼鏡、掃地機、白色家電)、車載電子。

## 核心競爭優勢
1.  **技術領先與多元佈局**：積極投入Wi-Fi 7/8、5G模組及多種先進數位影像模組（如3D、ToF、eUSB鏡頭）的研發，涵蓋無線通訊與AI視覺兩大核心技術。
2.  **市場滲透廣泛**：產品應用從PC擴展至遊戲機、印表機、車載、無人機、機器人及白色家電等多元領域，降低單一市場風險。
3.  **AIoT先機**：搭上AI熱潮，積極發展AI無線模組、AI相機模組及機器人AI視覺辨識，搶佔邊緣運算及物聯網終端設備的商機。
4.  **母公司和碩資源整合**：作為和碩子公司，在供應鏈管理、客戶資源與技術支援方面具備潛在優勢，特別是在消費性電子與PC相關應用。
5.  **既有市場領導地位**：無線模組在筆電市場市佔率約五成，數位影像模組在筆電攝像頭模組產業位列前三名，具備穩固的客戶基礎與規模經濟效益。

## 產業分析

### 市場規模與供需狀況
海華科技所在的無線通訊模組與數位影像模組市場正迎來快速增長，主要驅動力來自AI、IoT與5G/Wi-Fi 7等新興技術應用。

*   **無線通訊模組**：
    *   全球市場規模：2024年人民幣**436億元**，預計2029年達人民幣**726億元**，2025-2029年複合年增長率為**10.6%**。
    *   Wi-Fi模組市場：2024年**41.5億美元**，預計2025年增至**45.2億美元**，2034年達**117.5億美元**。
*   **數位影像模組**：
    *   全球相機模組市場：2025年**531.2億美元**，預計2034年達**1332.6億美元**，2026-2034年複合年增長率為**10.76%**。
    *   3D影像與感測市場：2020年**68億美元**，預計2026年達**150億美元**，複合年增長率達**14.5%**。
*   **物聯網設備數量**：預計2024年底增長**13%**至**188億台**，2030年達**400億台**。
*   **端側AI市場**：預計2025年人民幣**3219億元**，增長至2029年人民幣**1223萬億元**，複合年增長率高達**39.6%**。
*   **供需狀況**：受AI拉動，物聯網設備與端側AI需求爆發，模組市場整體呈現需求快速增長的趨勢，高階模組供不應求。

### 產業平均毛利率水準
模組製造業的毛利率受產品複雜度、市場競爭及技術領先程度影響。高階、具備新技術（如Wi-Fi 7、AI功能、高速光模組）的產品，其毛利率通常優於傳統模組。例如，立積受惠Wi-Fi 7出貨比重拉升，2025年Q2毛利率提升至**35.4%**；Semtech物聯網部門2024年毛利率為**39%**。海華利潤率的提升也印證了其產品組合優化的成果。

    *   **機會**：海華在多種數位影像模組的布局，使其能提供更精準、更可靠的視覺解決方案，特別是在自動駕駛艙、機器人導航等高階應用。
