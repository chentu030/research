# 3673 TPK-KY 深度研究報告

## 一句話摘要
TPK-KY作為觸控技術領導廠商，正積極轉型至高毛利利基市場，如3D列印、車用光達，並透過策略性投資奕力科技跨足半導體顯示驅動IC領域，以擺脫傳統觸控產業限制，儘管營運仍面臨市場逆風，但新事業佈局為長期成長帶來動能，2026年合併奕力營收後，獲利能力有望逐步改善。

## 公司概覽
TPK-KY宸鴻科技集團是多點投射式電容觸控技術的開創者，具備垂直整合的生產製程，提供1至110吋全方位一站式的觸控技術應用解決方案。

*   **主要業務與核心產品：**
    觸控感應器、觸控模組、觸控螢幕、ITO玻璃及保護玻璃相關產品的研發、生產及銷售。此外，公司也以3D列印方式生產消費性產品之主要元件。
*   **終端應用：**
    穿戴式裝置、智慧聯網裝置、電子書(eReader)、智慧型手機(7吋以下)；平板電腦(7-11吋)；大尺寸平板電腦、2-in-1電腦、筆記型電腦(11-16吋)；All-in-One PC、車用中控螢幕、互動式電子白板(16吋以上)。
*   **製造基地：**
    中國福建廈門設有9個製造基地。2023年7月已於泰國成立子公司TPK Auto Tech Thailand Co., Limited，並準備建立第二個生產基地。
*   **產品營收結構 (2024年資料)：**

| 產品類型           | 2024年營收佔比 | 備註                                   |
| :----------------- | :------------- | :------------------------------------- |
| 11至16吋產品       | 49%            |                                        |
| 7至11吋平板產品    | 32%            |                                        |
| 7吋以下小尺寸產品  | 7%             | 相較2023年下降60%                      |
| **主要產品及營業比重 (通用)** | **90.93%**     | 觸控模組 (資料出表時間2025-11-21)      |

## 核心競爭優勢
1.  **觸控技術領先地位：** 作為多點投射式電容觸控技術的開創者，具備深厚的技術積累和垂直整合生產製程，能提供1至110吋全方位觸控解決方案，有助於縮短客戶產品開發週期。
2.  **多元化應用布局：** 產品應用廣泛，從消費性電子延伸至車用、工控、電子書、互動式電子白板等利基市場，降低單一市場波動風險。
3.  **新興高毛利事業轉型：** 積極投入3D列印（應用於鞋中底、護墊、運動用品）、車用光達(LiDAR)等高附加價值領域，並透過收購奕力科技股權，策略性跨足半導體顯示驅動IC（DDIC）市場，精準瞄準AI與顯示應用趨勢，有助於優化產品組合與提升獲利能力。
4.  **彈性生產與地緣風險分散：** 在中國廈門設有多個製造基地，並在泰國建立第二個生產基地，以降低地緣政治風險並滿足客戶對非中國產能的需求。

## 產業分析

### 產業數據
*   **全球市場規模與CAGR成長率**
    *   2025年全球觸控螢幕市場規模已達**1,572.2億元人民幣**，預計到2032年將突破**2,052.5億元人民幣**，複合年增長率（CAGR）為**3.9%**。
    *   2026年全球觸控螢幕顯示器市場規模為**1,126.1億美元**，預計到2034年將增長至**3,189.9億美元**，2026年至2034年複合年增長率為**13.90%**。
    *   商用觸控顯示器市場：2025年市場規模**64.1億美元**，預計到2035年將突破**144.9億美元**，2026-2035年CAGR超過**8.5%**。
    *   2025年全球顯示面板產值預估達**1,265億美元**。Omdia預計，2026年全球顯示面板總面積需求預計較去年同期成長**6%**。

*   **供需狀況**
    *   **整體顯示器面板市場：** 2025年全球顯示器面板市場邁入「存量競爭」新階段，總出貨量約**1.64億片**，同比增長**2.1%**，但增速同比顯著收窄。預計2026年出貨**163.4百萬片**，同比微降**0.4%**。
    *   **手機面板：** TrendForce預估，2026年全球手機面板出貨量約**21.4億片**，較2025年下滑**7.3%**，轉為年減態勢，主要受記憶體缺貨漲價衝擊。
    *   **稼動率：** 目前面板產業稼動率維持在**80%到85%**之間，行業龍頭控產保價已成共識。

*   **產業的平均毛利率水準**
    觸控面板和相關顯示器產業的毛利率水準差異較大。TPK-KY 2025年Q3毛利率為**7.28%**。台灣同業全台（3038）2025年Q3毛利率為**22.68%**。義隆電子（2458）2025年全年毛利率為**48.6%**。

### 產業趨勢
1.  **內嵌式觸控技術普及：** In-cell、On-cell等技術將觸控感測器整合於顯示面板，具有更薄、輕、成本低等優勢。此技術在智慧型手機市場佔主導地位，並擴展至平板電腦。
    *   **具體影響：** 擠壓傳統外掛式觸控面板廠商的市場空間，迫使TPK-KY等廠商轉型或專注利基市場。
2.  **大尺寸化與利基市場應用：** 觸控面板應用尺寸擴大至大尺寸商用顯示器、車載中控螢幕、互動式電子白板，以及醫療、工控等高規格、客製化市場。
    *   **具體影響：** 大尺寸化推動商用觸控顯示器市場CAGR超過**8.5%**。車載觸控需求持續增長，TPK-KY在這些市場的布局可望受惠。
3.  **柔性與新顯示技術：** 柔性/可摺疊顯示、Mini/Micro-LED、電子紙等新技術為觸控面板帶來設計與應用新機會。
    *   **具體影響：** TPK-KY若能在摺疊螢幕等高階市場取得技術突破，將有助於開闢新機遇。
