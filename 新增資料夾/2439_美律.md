# 2439 美律 深度研究報告

## 一句話摘要
美律 (2439) 2025年營收創新高，但受匯率逆風、產能移轉成本及關稅分攤影響，獲利承壓。展望2026年，高階耳機、醫療產品及新併購案將推動營收個位數成長，最關鍵的成長動能來自切入美系折疊手機供應鏈，預計下半年開始出貨，搭配東南亞產能擴張，中長期營運表現可望谷底回升，但短期獲利仍需觀察費用控管效益。

## 公司概覽
美律實業（2439）為電聲元件領導廠商，專注於電聲產品的研發、製造與銷售，是全球前十大手機揚聲器供應商之一。其產品廣泛應用於消費性電子及醫療領域。

**產品線與成長動能：**
美律的核心產品涵蓋耳機 (Headset)、揚聲器 (Speaker)、麥克風、電池模組以及醫療聽力產品（如助聽器和睡眠耳機）。2026年主要成長動能預計來自高階頭戴式耳機、Audio耳機、電競耳機以及醫療產品。

**營收結構 (2025年第四季)：**
| 業務類別   | 營收佔比 | 年增率 (YoY) | 成長動能/備註                                                                                                     |
| :--------- | :------- | :----------- | :---------------------------------------------------------------------------------------------------------------- |
| **耳機 (Headset)** | 67%      | +28%         | 最大營收來源。其中Audio耳機佔比64% (年增28%，受中國客戶需求帶動)；電競耳機佔比22% (年增15%)；TWS耳機佔比14% (年減34%，新舊產品交替影響)。 |
| **揚聲器 (Speaker)** | 26%      | -3%          | 手機用揚聲器佔比66% (年增9%)；電腦用揚聲器年減21% (同業競爭及客戶無新品)。                                                    |
| **其他 (Others)**  | 7%       | -5%          | 包含電池、麥克風及醫療產品等。電池新客戶專案延遲。                                                                        |

**製造基地與產能布局：**
美律的製造基地分佈在中國、越南和泰國。
*   **2025年**：中國廠約佔總產能的60%，非中國地區（越南與泰國）合計佔比約為40%。
*   **22026年目標**：越南與泰國的產能合計將進一步提升至50%以上，與中國產能各佔一半，以分散地緣政治風險並配合客戶需求。
*   **泰國二廠**：已於2025年第四季開始投產，將成為美律海外出貨的主力基地之一。

**所屬細分產業與題材：**
電聲元件及通信網路業。相關題材包括：高階頭戴式耳機、電競耳機、Audio耳機、美國OTC助聽器市場、折疊手機供應鏈（市場推測為蘋果）、電池產品、電腦用揚聲器、永續發展（ESG）、日本MWT併購案。

## 核心競爭優勢
1.  **多元化產品組合與技術深度：** 美律擁有廣泛的電聲產品線，涵蓋耳機、揚聲器、麥克風、電池模組及醫療聽力產品，顯示其在聲學技術上的深厚積累與多元佈局。
2.  **高階市場與新興應用拓展：** 積極佈局高階頭戴式耳機、電競耳機和Audio耳機等高利潤市場，並成功切入美國OTC助聽器市場，抓住產業結構性成長機會。
3.  **大客戶供應鏈地位：** 已成功打入美系客戶折疊手機供應鏈（市場推測為蘋果），並長期為蘋果iPhone的聲學元件供應商之一，顯示其在頂級品牌客戶中的重要地位與品質認可。
4.  **彈性且多元的全球產能布局：** 策略性地將產能從中國轉移至越南與泰國，目標2026年非中國產能佔比達50%，不僅有助於降低關稅與匯率風險，更能提供客戶供應鏈韌性與彈性。
5.  **透過併購擴展市場與技術：** 2026年2月完成日本MWT併購案，此舉有助於美律拓展日本市場，切入知名耳機品牌，帶來額外的營收成長動能。

3.  **無線化與連線技術進步：** 藍牙5.0/5.3、Wi-Fi 6等技術進步推動高品質音訊串流和智能家居整合，促進無線耳機與揚聲器市場成長。無線耳機市場預計2026年將突破7.2億副，市場規模約1,200億美元。美律在此趨勢下需持續提升無線產品技術與品質。
