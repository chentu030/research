# 8438 昶昕 深度研究報告

**生成日期：2026年03月06日**

## 一句話摘要

昶昕（8438）作為電子特用化學品與金屬回收再生業者，以獨特的循環經濟模式服務PCB及半導體產業。受惠AI需求推升高階化學品用量及國際銅價上漲，營運動能強勁，2025年全年營收達38.42億元。隨著越南新廠預計於2026年第一季試量產，以及台灣廠區新增溶銅裝置，公司成長動能將從前端高階化學品與後端再生銅鹽雙軌推進，有望在東南亞市場擴張和全球供應鏈重組中搶佔先機。

## 公司概覽

昶昕股份有限公司（8438）成立於1991年，是一家專注於為電子產業提供特用化學品製造與金屬回收再利用的廠商，其業務模式獨特，採行循環經濟理念。

*   **公司主要做什麼？核心產品/服務是什麼？**
    *   **特用化學品（前段業務）**：主要為電子產業提供線路成型所需的化學品，應用於印刷電路板（PCB）、載板、封裝及半導體等領域，例如蝕刻液、剝錫液等。昶昕是台灣唯一同時具備蝕刻液與剝錫液生產能力，並持有政府核發的廢液回收處理執照的廠商。
    *   **資源回收再生產品（後段業務）**：從客戶端購回使用後含有高價值金屬（如銅、錫）的廢液，透過資源再生技術，將非金屬部分再製成新的特用化學品，而金屬部分則提煉成高純度的硫酸銅等銅鹽產品，並銷售至礦業及其他工業應用。

*   **製造基地**

    *   **中國大陸廠區**：友緣崑山。
    *   **越南廠區**：友緣越南新廠，預計於2025年底至2026年第一季期間進行試量產，以因應AI產品對電子線路蝕刻精密需求的提升。
    *   **泰國市場**：正考慮布局，可能由華帶廠供貨，以跟進客戶的全球化布局。

## 核心競爭優勢

1.  **獨特且完整的循環經濟模式**：昶昕是台灣唯一同時具備蝕刻液與剝錫液生產能力，並持有政府核發廢液回收處理執照的廠商。其從前端特用化學品供應到後端廢液回收、金屬再生利用的垂直整合模式，不僅降低了原材料成本，也符合全球ESG與環保趨勢，形成高進入門檻的競爭優勢。
2.  **高階電子產業應用能力**：產品廣泛應用於PCB、載板、封裝及半導體等高階電子領域，受惠於AI伺服器、HPC等對精密電子線路蝕刻需求日益提升，具備提供高附加價值產品的能力。
3.  **靈活的原料來源與成本控制**：透過廢液回收取得再生銅，並增設溶銅裝置可外購銅料進行加工，形成「雙軌銅源策略」。這不僅能應對國際銅價波動，也確保了在電子業廢液回收量不足時的產能彈性，有助於維持穩定的毛利。
4.  **區域擴張策略與供應鏈韌性**：積極布局越南新廠及評估泰國市場，以因應全球供應鏈重組及客戶海外設廠需求，強化東南亞市場的供貨體系，提升公司全球競爭力與營運韌性。

## 法說會重點

昶昕於2025年8月14/16日及11月19日舉行法人說明會，管理層傳達以下重點：

*   **產品線展望**：
    *   **後段產品（再生銅鹽）**：2025年第四季訂單能見度佳，市場對銅的需求強勁，主要受惠於基礎建設、電動車等應用。訂單呈現供不應求狀態，公司已新增溶銅裝置，可外購銅料進行加工以增加產能彈性。
    *   **前段產品（特用化學品）**：2025年上半年訂單表現良好，下半年需觀察電子業景氣。AI伺服器相關應用佔比逐漸提升，需求表現優於預期。
*   **產能與資本支出**：
    *   **越南新廠**：建設進展順利，已於2025年第四季完成機台設備裝機，規劃於2026年第一季導入試量產，初期將穩健、分階段提升產能，預計2026年第一至第二季逐步開出，成為新的成長動能。
    *   **綠能建設**：持續推動廠內綠能設施，預計2026年第二季前將有新增綠能產出，有助於降低碳排與符合ESG趨勢。
    *   **資本支出**：董事會於2026年2月24日決議發行總額3億元的國內第一次無擔保轉換公司債，以充實營運資金，支持未來的擴產與營運發展。
*   **管理層 Guidance**：
    *   **2025年第四季**：抱持審慎樂觀態度。
    *   **2026年**：期待東南亞（特別是越南）的成長性，並看好台灣電子業的發展。
