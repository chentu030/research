# 6533 晶心科 深度研究報告

**今天日期：2026年03月06日**

## 一句話摘要
晶心科作為全球 RISC-V 矽智財（IP）領導者，正透過高階 Cuzco 系列處理器與 AndesAIRE 解決方案，積極搶攻 AI、資料中心及車用電子等高成長市場。儘管 2025 年受研發費用與市場逆風影響面臨短期虧損，但隨著 RISC-V 生態系統的成熟與多項授權案陸續進入量產，以及 Cuzco 系列產品預計於 2026 年中開始貢獻營收，公司有望在 2026-2027 年迎來權利金收入的爆發性成長與獲利轉折。

## 公司概覽
晶心科（6533）是一家專注於 CPU IP（矽智財）的供應商，也是 RISC-V 國際組織的創始頂級會員，在全球 RISC-V 生態系統中扮演領導角色。公司採無晶圓廠（Fabless）模式，總部設於台灣新竹科學園區，並於美國德州奧斯汀設有子公司 Condor Computing 專責高階 RISC-V CPU IP 開發。

**公司主要業務與核心產品/服務：**
*   **AndeStar™ 架構**：專利的32位元/64位元RISC風格CPU架構。
*   **AndesCore™ 系列CPU**：涵蓋32位元/64位元處理器，針對不同應用領域：
    *   **AX系列**：高效能應用，支援Linux，適用於AI、資料中心，如AX65系列、AX66、AX46MPV。
    *   **D系列**：數位訊號處理（DSP）應用，適用於AIoT、消費性電子，如D25F-SE、D45-SE、D23-SE。
    *   **N系列**：主流嵌入式應用，適用於車用與微控制器（MCU），如N25F-SE。
*   **AndesAIRE™ 解決方案**：專為邊緣及端點推論設計，例如最新的深度學習加速器（DLA）AnDLA I370。
*   **AndeShape™ 平台**、**AndeSight™ IDE** (整合開發環境)及**AndeSoft™ SW Stack** (軟體堆疊)。
*   **最新高階產品**：代號「Cuzco」的CPU系列，鎖定資料中心、AI及車用高效能運算市場，預計於2026年第二季/第三季開始貢獻營收。

## 核心競爭優勢
1.  **RISC-V 生態系統領導地位**：晶心科是 RISC-V 國際組織的創始頂級會員，在標準制定、技術發展及生態系推動上扮演關鍵角色，擁有超過 30% 的 RISC-V 晶片出貨量市場份額，受益於 RISC-V 開放、高效能、低功耗及可客製化的特性，在 AI、車用和 IoT 市場具備競爭優勢。
2.  **高階 CPU IP 技術領先**：專注於中高階 RISC-V CPU 核心開發，如針對資料中心和 AI 的 Cuzco 系列以及 AX 系列高效能處理器，具備向量處理器與客製化擴展（ACE），已獲大型美系客戶青睞，技術實力與前瞻佈局為其提供差異化優勢。
3.  **深耕高成長應用市場**：策略性將資源投入 AI、車用電子與物聯網（IoT）等高成長領域，並已取得 ISO 26262 功能安全認證，產品進入多家國際車用客戶供應鏈，形成早期進入者優勢。
4.  **全球研發與業務佈局**：在美國奧斯汀設立 Condor Computing 子公司，專責超高效能 RISC-V CPU IP 開發，並在德國慕尼黑設立技術資源中心，有效強化全球研發能量並深耕重要市場客戶。
5.  **穩固的權利金成長潛力**：隨著近年簽訂的 RISC-V 授權案陸續進入量產，權利金收入預計在未來 2-3 年內大幅提升，將優化營收結構並帶來更穩定的現金流。

**利潤率變化原因分析：**
*   **高毛利率維持**：晶心科的毛利率長期維持在極高水準 (接近 100%)，突顯其 IP 產品的高附加價值特性，成本主要為研發費用。
*   **營業費用增加**：2025年上半年營業費用年增約 6%，其中研發（R&D）支出增加比例最高，主要因 2024年新增人力於 2025年需支付全年薪酬，以及美國研發及業務團隊擴張和德國子公司設立等開銷。
*   **上半年虧損擴大**：2025年上半年因市場逆風、美國新產品開發費用增加及匯兌損失，導致單季虧損較大 (2025年Q2 EPS -5.58元)。累計前三季營業損失達新台幣 361,177仟元，較去年同期虧損增加 14.0%。
*   **第三季營運回溫**：2025年第三季營運顯著回溫，營收季增 **66.9%**，單季 EPS 轉正為 **0.17元**，稅後淨利率達 **1.81%**，營業利益率虧損幅度亦大幅收斂至 -5.16%。公司自 Q3 起已加強費用控管。
*   **業外收支波動**：2025年第二季因匯兌損失造成單季虧損擴大，業外收支對營收比率在 2025年 Q2 為 -19.68%，Q3 為 -7.82%，顯示業外支出或損失仍對獲利產生負面影響。
