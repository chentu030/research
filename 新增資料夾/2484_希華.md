# 2484 希華 深度研究報告

## 一句話摘要
希華（2484）作為頻率控制與計時元件領導廠商，正積極轉型高附加價值的AI、5G、車用電子及低軌衛星市場，儘管2025年受產品價格與匯率壓力導致獲利承壓，但隨著產業需求回溫及一次性業外收益挹注，2026年營運展望審慎樂觀，有望迎來本業與獲利雙重成長機會。

## 公司概覽
希華晶體科技股份有限公司（2484）成立於1988年，為全球領先的頻率控制與計時元件製造商，產品線涵蓋從人造石英棒到各類石英晶體與振盪器，廣泛應用於網通、物聯網、PC/消費性電子、手機及車用等多元領域。公司透過自主研發與垂直整合能力，提供客戶高精度、小型化的解決方案。

**主要產品與服務：**
*   **原料與晶片：** 人造石英、音叉晶片 (Tuning Fork Wafer)。
*   **石英晶體 (Crystal)：** 包含Seam Seal、32.768 KHz 音叉晶體，以及應用於手機的TSX產品。
*   **石英振盪器 (Oscillator)：** 包含一般石英振盪器 (OSC)、溫補型石英振盪器 (TCXO)、壓控型石英振盪器 (VCXO) 及壓控溫度補償型石英振盪器 (VCTCXO) 等。

**營收結構（依應用領域，2025年法說會簡報資料）：**

**製造基地：**
希華在台灣設有竹南廠（光刻晶片）、台中總部（研發及主要生產）及台南廠（石英晶棒、音叉KHz產品）。

## 核心競爭優勢
1.  **完整垂直整合能力：** 從石英晶棒長成到晶片設計製造，希華具備垂直整合能力，有效掌握成本與品質，提供多樣化產品線滿足客戶需求。
2.  **技術領先與小型化趨勢：** 積極投入小型化、高頻率、高精度產品研發，如1008超小型化元件與TCXO/VCTCXO等高階振盪器，符合5G、AIoT及穿戴裝置輕薄短小的市場趨勢。
3.  **多元應用市場佈局：** 營收應用廣泛涵蓋網通、物聯網、PC、手機及車用，有效分散單一產業風險，並積極拓展高毛利車載及低軌衛星應用。
4.  **穩健的財務結構：** 負債比率維持在健康水平，顯示公司財務管理得宜，為未來擴張與轉型提供良好基礎。

### 產業趨勢
1.  **關鍵技術趨勢與影響：**
    *   **小型化、高頻化與高精度化：** 5G、AIoT、穿戴裝置等應用推動石英元件朝更小尺寸、更高頻率穩定性發展。希華需持續投入研發，提升小型化、高頻率TCXO/VCXO等產品生產能力，以滿足市場需求。
    *   **車用電子與工業控制嚴苛要求：** 電動車、自動駕駛、工業4.0對元件的耐溫、耐震、可靠性提出更高要求。希華積極開發符合AEC-Q200車規標準的產品，將有助於進入高附加價值市場。
    *   **MEMS振盪器的崛起：** MEMS振盪器在小型化、抗震性具潛在優勢。儘管石英晶體在精度與穩定性仍領先，但MEMS的進步對希華構成潛在競爭與合作機會。

2.  **對希華而言的機會與威脅：**
    *   **機會：** 5G/6G通訊、AIoT、電動車市場的擴張帶動高頻、高穩定度石英元件需求。技術升級與產品組合優化（如1008超小型化元件）有助於提升附加價值。供應鏈重組趨勢亦可能為台灣廠商帶來轉單機會。
    *   **威脅：** 全球經濟波動與產業景氣循環影響訂單。日本、中國競爭加劇導致價格壓力，影響毛利率。MEMS振盪器的發展帶來替代性競爭。原物料價格波動可能影響生產成本。

3.  **相關投資題材連結：**
    *   **AI (人工智慧)：** AI伺服器、邊緣運算設備需高穩定度、低抖動石英振盪器提供精準時脈。希華的高精度振盪器產品將受益於AI應用普及。
    *   **電動車：** 電動車電子化程度高，從ADAS到動力系統，皆需大量符合車規標準的石英元件。希華積極轉型車載市場，將從中獲益。
    *   **5G/6G通訊：** 5G/6G基地台與終端設備對頻率精度要求極高，高頻、低相位雜訊的TCXO/OCXO不可或缺。希華網通業務佔比高，直接受益於通訊技術演進。
    *   **低軌衛星：** 希華轉投資Rakon在低軌衛星等高階技術應用具價值，未來有望通過合作夥伴切入衛星通訊市場。
