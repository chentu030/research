# 2345 智邦 深度研究報告
## 一句話摘要
智邦（2345）受惠於AI基礎建設投資熱潮，800G高階交換器及AI加速卡模組出貨動能強勁，搭配新客戶訂單挹注與全球產能擴張，預計2026年營收與獲利將創歷史新高，成為AI網通浪潮下的主要受惠者，具備長線成長潛力。

## 公司概覽
智邦科技（2345）為全球領先的網路通訊設備製造商，專精於乙太網路和無線設備的研發、設計與製造，提供OEM/ODM服務，並透過SMC與Edge-Core雙品牌拓展市場。公司是全球白牌交換器（White-box Switch）的龍頭廠商之一，受益於雲端服務供應商（CSP）對開放式網路架構的需求。

**核心產品線：**
*   **網路交換器 (Network Switches)**：提供100G、200G、400G、800G高速交換器，並積極布局1.6T技術。為AI資料中心和雲端服務供應商擴建的關鍵設備。
*   **AI加速卡模組 (AI Accelerator Modules)**：主要供應大型雲端服務商，在AI加速卡需求持續攀升下，是智邦的重要成長動能。
*   **網路應用設備 (Network Application Equipment)**
*   **網路接取設備與無線網路設備 (Network Access Equipment & Wireless Network Equipment)**

**營收結構（2024年第三季資料）：**

| 產品線                     | 營收佔比 |
| :------------------------- | :------- |
| 網路交換器                 | 57.6%    |
| 網路應用設備及AI加速卡     | 33.8%    |
| 網路接取設備與無線網路設備 | 5.5%     |

**製造基地與銷售區域：**
智邦在全球多地設有製造基地，包括台灣（竹北廠）、越南、美國（Irvine/德州）、馬來西亞、中國、杜拜、土耳其，以應對地緣政治變化及客戶分散生產需求。目前美洲地區為主要銷售市場，2024年前三季佔比達72%。

## 核心競爭優勢
1.  **白牌交換器市場領導地位**：作為全球白牌交換器龍頭，智邦能直接供應超大規模雲端服務商(Hyperscalers)，有效掌握市場趨勢與需求。
2.  **AI基礎建設核心供應商**：智邦的高階800G交換器及AI加速卡模組是AI資料中心不可或缺的基礎設備，直接受惠於AI需求爆發。
3.  **先進技術布局**：積極投入1.6T交換器、共同封裝光學(CPO)、光波長交換(OWS/WSS)及機櫃級整合(L11)等前瞻技術，確保未來競爭力。
4.  **全球化產能布局**：透過在台灣、越南、美國、馬來西亞等多地擴產，分散地緣政治風險，並強化供應鏈韌性，滿足客戶彈性需求。
5.  **與主要雲端服務商深度合作**：與Meta、AWS等大型雲端業者維持緊密合作關係，並持續開拓新客戶，確保訂單能見度與出貨量。

## 法說會重點 (2025年11月7日)
*   **總經理李訓德表示**：公司將全力滿足客戶需求，並審慎樂觀看待2026年營收與獲利成長，強調AI需求並非泡沫，客戶資本支出持續強勁。
*   **產品線展望**：
    *   **網路交換器**：2026年最大成長來源。雲端服務商加速將交換器規格從400G升級至800G，智邦取得多項新客戶專案。800G高階交換器已大規模放量。預估2026年新客戶將貢獻約**15%**營收，使網路交換器營收比重提升至**48%**。
    *   **AI加速器模組**：持續擴大，儘管供應鏈導入新供應商，但智邦仍掌握大部分訂單，主要客戶（如AWS）訂單持續成長。預計2025年亞馬遜加速卡佔比將來到**40-45%**。新世代AI加速器規格提升，有望提高產品平均售價。
    *   **1.6T產品**：預計於2026年下半年量產，智邦目標成為市場首批出貨者。
*   **訂單能見度**：主要客戶Meta已將網通產品訂單提前至2026年第一季出貨。AWS維持強勁資本支出，確保智慧網卡與AI加速卡模組出貨。
*   **產能與資本支出**：2025年Q3存貨為318億元，主要為Q4出貨準備。2026年主要挑戰在於產能、人才與關鍵元件短缺，故積極進行全球擴產。AI加速卡2025年月產能約**20-25萬顆**，並將逐漸提升。
*   **2026年第一季Guidance**：營收將呈現季減個位數百分比，但年成長仍維持高檔，呈現「淡季不淡」的韌性。

### 產業趨勢
1.  **AI驅動的高速網路升級 (800G 及 1.6T 交換器)**：
    *   **趨勢**：生成式AI、雲端運算導致頻寬需求從Gbps級躍升至Tbps級。800G交換器已進入快速成長期，2025年滲透率預估達**45%**，市場規模上看**3.5億美元**。1.6T交換器預計2026-2027年導入。
    *   **影響**：智邦的高速交換器直接受益，是AI模型訓練和推論的關鍵基礎設施。

2.  **AI資料中心光互連技術的演進 (矽光子與共封裝光學 CPO)**：
    *   **趨勢**：AI資料中心光互連市場2025年達**99.4億美元**，預計到2033年達**310.4億美元**，CAGR為**15.3%**。矽光子與CPO技術能提高傳輸速度、降低延遲和功耗。
    *   **影響**：光互連技術是AI及高效能運算（HPC）應用關鍵，智邦積極投入此領域以滿足未來高速互連需求。

3.  **資料中心架構的重塑 (液冷技術與高密度機架)**：
    *   **趨勢**：AI運算導致資料中心面臨極端機架密度挑戰（2026年單機架負載達100-200 kW），推動液冷技術成為主流，機櫃級AI平台帶動散熱與電源供應方案升級。
    *   **影響**：智邦積極布局L11機櫃級整合方案，切入高密度AI運算平台的關鍵組件供應。

**對智邦的具體機會和威脅：**
*   **機會**：
    *   AI資料中心需求爆發，尤其Hyperscalers對800G交換器的加速導入。
    *   新增Hyperscaler客戶與多項ASIC相關專案，預計2026年貢獻**15%**營收。
    *   產品組合優化（網路交換器佔比提升至2026年**48%**）與AI加速卡模組ASP提升，將推升毛利率至2026年預估**18.9%**。
    *   白牌交換器龍頭地位，受益於ODM直接銷售模式增長。
*   **威脅**：
    *   市場競爭加劇，尤其AI加速器模組業務雖仍具優勢，但新供應商導入可能影響增速。
    *   技術快速迭代風險，未能及時跟上1.6T等高速技術發展可能被邊緣化。

**相關投資題材：**
*   **AI (人工智慧)**：AI訓練和推論對資料傳輸頻寬和速度的極高要求，直接拉動了對智邦高速網路交換器（800G、1.6T）和AI加速卡模組的需求。
*   **高速運算與資料中心**：隨著全球數位基礎設施擴展，智邦的產品是資料中心不可或缺的基礎設施。
