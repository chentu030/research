# 1446 宏和 深度研究報告

## 一句話摘要
宏和 (1446) 已成功轉型為以營建開發為核心的資產股，營建業務於2024年貢獻高達97%的營收，帶動全年EPS達9.82元。儘管近期紡織業務面臨市場逆風導致營收波動劇烈，且財務體質需持續關注，但公司管理層承諾穩健經營，並祭出2025年高達8.68%的現金殖利率，凸顯其資產價值與股東回報潛力。

## 公司概覽
宏和精密紡織股份有限公司（1446）成立於1978年，1991年上市。其核心業務已從傳統紡織製造大幅轉向營建地產開發，近年營收主要來自營建專案認列。

**主要業務與產品線：**
*   **紡織布料與織品製造：** 生產針織布、梭織布、功能性紡織品（抗菌、防水、透氣）、環保與可持續發展產品（回收再利用材料、生物基纖維）。
*   **營建及地產業務：** 國民住宅及商業大樓的興建、出租與出售。

**營收結構 (2024年為例)：**

| 業務類別   | 營收佔比 |
| :--------- | :------- |
| 營建事業部 | 97%      |
| 平織布業務 | 3%       |

*註：根據2025年11月法說會資訊。*

**製造基地：**
宏和總部及主要生產基地設於**台灣**，為技術創新與研發核心。此外，亦在**中國大陸**與**東南亞**設有生產據點。目前未有各基地具體營收貢獻比例資料。

**所屬細分產業與相關題材：**
公司隸屬於**紡織纖維產業類別**，同時涉足**營建地產**業務。相關題材包括：
*   **織布與紡織中游**
*   **資產股**：因其營建地產業務及可能持有的不動產資產。
*   **功能性紡織品**：受惠於市場對高功能、環保紡織品的需求。

## 核心競爭優勢
1.  **成功的業務轉型與資產活化：** 宏和成功將營運重心轉向營建開發，2024年營建事業部貢獻達97%的營收，大幅提升公司獲利能力，證明其在不動產開發與資產活化方面的執行能力。
2.  **穩健的財務策略：** 2025年11月法說會管理層強調「穩定成長」及「穩健經營」，計劃在維持健全財務結構基礎上保守穩健開發不動產，降低盲目擴張風險。2024年公司負債比率顯著降低，股東權益大幅增加。
3.  **高殖利率吸引力：** 2025年宣布配發現金股利2.00元，現金殖利率高達8.68%，對價值型投資者具吸引力。
4.  **紡織業務的特定客戶基礎：** 其平織布業務客戶群廣泛，包含日本知名布疋商社、量販店、品牌專賣店等，顯示在特定市場仍具一定競爭力。

## 法說會重點
宏和於2025年11月25日受邀參加元大證券線上法人說明會，闡述公司營運狀況，重點摘要如下：
*   **營運重心轉變：** 公司營運重心已從傳統紡織製造業大幅轉向營建開發。
*   **2024年財務表現卓越：** 營收實現爆炸性成長，稅後淨利及每股盈餘（EPS）大幅躍升，負債比率顯著降低，股東權益大幅增加。
*   **營建事業部主導獲利：** 2024年（113年）的優異財務表現，主要歸因於營建事業部的成功貢獻，其在2024年幾乎佔據了全部營收。
*   **未來營運策略：** 強調「穩定成長」及「穩健經營」。
*   **不動產開發計畫：** 計劃在維持健全財務結構的基礎上，保守穩健地開發不動產並進行投資合建。
*   **營收結構：** 法說會中指出，營建事業部貢獻公司約97%營收，平織布業務佔3%。

*   2025年約為**1.065兆美元**至**2.123兆美元**之間。
*   多份報告預估2026年至2034年間複合年增長率（CAGR）介於**5.09%至7.11%**。
*   機能性紡織品市場：預計從2025年的**1,598億美元**成長到2026年的**1,715.5億美元**，CAGR為**7.4%**，並預計2030年達到**2,308.1億美元**，CAGR為**7.7%**。

### 產業趨勢
1.  **數位化轉型與智慧製造：**
    *   **具體影響：** AI貫穿設計、生產、物流、銷售全鏈條，實現智能排產、質量預警、個性化定制，可降低返工率30%以上，縮短交付週期20%-30%。智能驗布、自動化倉儲等普及，使小批量、快交付成為主流。
2.  **綠色低碳與永續發展：**
    *   **具體影響：** 歐盟CBAM將於2026年實施，促使紡織業推動低碳永續、循環利用，再生纖維、生物基纖維產能擴張，綠色產品認證提升。對環保、可追溯性和透明供應鏈的需求日益增長。
3.  **高值化與機能性紡織品：**
    *   **具體影響：** 產業價值鏈向中高端攀升，龍頭企業聚焦技術研發與品牌建設，功能性面料、智能可穿戴、高端產業用紡織品佔比提升。工業/技術紡織品成長最快，受汽車輕量化及基礎設施現代化推動。

**對宏和的具體機會與威脅：**
*   **機會：**
    *   若能在機能性紡織品及環保可持續產品領域持續創新，將受惠於高值化市場成長。
